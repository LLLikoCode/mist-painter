# 玩家资源系统

> **系统:** Player Stats
> **重要性:** ⭐ 核心系统
> **设计理念:** 资源是游戏的节奏控制器

---

## 1. 资源概述

### 1.1 四大资源

| 资源 | 初始值 | 上限 | 用途 | 恢复方式 |
|------|--------|------|------|----------|
| HP (生命值) | 100 | 200 | 生存 | 休息/道具/技能 |
| SP (精神值) | 50 | 100 | 技能 | 休息/道具 |
| 墨水 | 50 | 100 | 迷雾绘制 | 收集/商店 |
| 体力 | 100 | 200 | 移动/动作 | 休息/食物 |

## 2. HP (生命值)

### 2.1 基础属性

```
初始值: 100
上限: 200 (随等级提升)
```

### 2.2 受伤来源

| 来源 | 伤害量 | 说明 |
|------|--------|------|
| 陷阱 | 10-30 | 根据陷阱类型 |
| 敌人攻击 | 10-50 | 根据敌人强度 |
| 环境伤害 | 5-20 | 毒气、尖刺等 |
| 饥饿 | 1/分钟 | 体力耗尽后 |

### 2.3 恢复方式

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 自然恢复 | 0 | 不可自然恢复 |
| 休息 | 5/分钟 | 安全区域 |
| 食物 | 10-30 | 消耗品 |
| 药水 | 30-50 | 消耗品，即时 |
| 技能 | 变化 | 特定技能 |

### 2.4 死亡处理

```
HP <= 0 时触发死亡:
1. 播放死亡动画
2. 保留部分进度
3. 丢失当前物品
4. 地图知识模糊化
5. 在检查点重生
```

## 4. 墨水系统

### 4.1 基础属性

```
初始值: 50
上限: 100
获取: 收集/商店
```

### 4.2 消耗机制

| 操作 | 消耗 | 说明 |
|------|------|------|
| 铅笔绘制 | 0.5/次 | 低消耗 |
| 羽毛笔绘制 | 1.0/次 | 中消耗 |
| 测绘仪绘制 | 2.0/次 | 高消耗 |
| 魔法卷轴 | 100% | 一次性消耗全部 |
| 绘制标记 | 0.2/次 | 放置地图标记 |

### 4.3 获取方式

| 方式 | 数量 | 说明 |
|------|------|------|
| 墨水瓶道具 | +30 | 常见道具 |
| 商店购买 | +20/次 | 花费金币 |
| 击败敌人 | 5-15 | 随机掉落 |
| 宝箱 | 20-50 | 随机数量 |

### 4.4 墨水不足处理

```gdscript
func try_paint(ink_cost: float) -> bool:
    if current_ink >= ink_cost:
        current_ink -= ink_cost
        ink_changed.emit(current_ink, max_ink)
        return true
    else:
        # 墨水不足，无法绘制
        return false
```

## 5. 体力系统

### 5.1 基础属性

```
初始值: 100
上限: 200 (随等级提升)
```

### 5.2 消耗机制

| 动作 | 消耗 | 说明 |
|------|------|------|
| 移动1格 | 1 | 基础消耗 |
| 奔跑1格 | 2 | 速度翻倍 |
| 绘制地图 | 5 | 每次绘制 |
| 仔细测绘 | 10 | 使用测绘仪 |
| 搜索/检查 | 8 | 发现隐藏要素 |
| 战斗 | 15-30 | 根据敌人强度 |

### 5.3 疲劳状态

| 体力百分比 | 状态 | 效果 |
|------------|------|------|
| 100-70% | 精力充沛 | 无惩罚 |
| 70-40% | 正常 | 无惩罚 |
| 40-20% | 疲劳 | 移动速度-10%，绘制误差+10% |
| 20-10% | 精疲力竭 | 移动速度-25%，绘制误差+25%，视野-1 |
| 10-0% | 濒死 | 移动速度-50%，无法绘制，视野-2 |
| 0% | 昏迷 | 需要救援或等待恢复 |

### 5.4 恢复方式

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 自然恢复 | 5/分钟 | 安全区域，不移动 |
| 休息 | 20/分钟 | 坐下休息，易受攻击 |
| 食物 | 10-50 | 消耗品 |
| 药水 | 50 | 稀有，有副作用 |
| 睡眠 | 100% | 只能在特定房间 |

## 6. 资源交互

### 6.1 资源影响关系

```
体力低 → 绘制误差增加
体力低 → 视野减少
SP低 → 技能威力降低
墨水不足 → 无法绘制
HP低 → 压力增加更快
```

### 6.2 资源平衡公式

```
单局资源消耗预期:
- 墨水: 200-300 (绘制消耗)
- 体力: 500-800 (移动消耗)
- HP: 50-100 (受伤)
- SP: 100-150 (技能)

单局资源获取预期:
- 墨水: 150-250 (收集)
- 食物: 5-10个
- 药水: 2-5个
```

## 7. 技术实现

### 7.1 数据结构

```gdscript
class_name PlayerStats
extends Node

# HP
var max_hp: float = 100.0
var current_hp: float = 100.0

# SP
var max_sp: float = 50.0
var current_sp: float = 50.0

# 墨水
var max_ink: float = 100.0
var current_ink: float = 50.0

# 体力
var max_stamina: float = 100.0
var current_stamina: float = 100.0

# 信号
signal hp_changed(current: float, max_hp: float)
signal sp_changed(current: float, max_sp: float)
signal ink_changed(current: float, max_ink: float)
signal stamina_changed(current: float, max_stamina: float)
signal player_died
signal ink_depleted
```

### 7.2 核心方法

```gdscript
## 受到伤害
func take_damage(amount: float) -> bool:
    if is_dead or is_invincible:
        return false
    set_current_hp(current_hp - amount)
    return true

## 消耗墨水
func consume_ink(amount: float) -> bool:
    var actual_cost = amount * ink_cost_multiplier
    if current_ink >= actual_cost:
        set_current_ink(current_ink - actual_cost)
        return true
    return false

## 消耗体力
func consume_stamina(amount: float) -> bool:
    if current_stamina >= amount:
        set_current_stamina(current_stamina - amount)
        return true
    return false

## 恢复体力
func restore_stamina(amount: float) -> void:
    set_current_stamina(current_stamina + amount)
```
