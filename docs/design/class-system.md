# 职业系统

> **系统:** Class System
> **重要性:** ⭐⭐ 进阶机制
> **设计理念:** 不同职业，不同体验

---

## 1. 系统概述

### 1.1 核心概念

玩家可以选择不同的职业开始游戏，每个职业有独特的起始装备、被动加成和技能倾向。

### 1.2 职业数量

共5种职业，每种职业适合不同的游戏风格。

---

## 2. 职业详情

### 2.1 测绘师 (Cartographer)

```yaml
职业描述: 专业的地图绘制师，精通制图技术

被动加成:
  - 绘图精度+30%
  - 记忆衰退速度-20%
  - 发现隐藏要素概率+15%

起始装备:
  - 测绘仪
  - 羊皮纸×5
  - 羽毛笔

技能倾向: 制图系
难度: ★★ (推荐新手)
```

## 4. 技能树系统

### 4.1 技能树结构

每个职业都可以学习所有技能，但有其倾向加成：

```
制图系 (测绘师加成):
├─ 精准测绘 (误差-10%)
│   └─ 快速绘图 (速度+30%)
│       └─ 记忆强化 (衰退-20%)
│           └─ 多图层绘制 (可同时看多层)
│               └─ 预知路径 (短暂显示出口方向)

生存系 (探险家/生存专家加成):
├─ 体能训练 (体力上限+20)
│   └─ 轻装上阵 (负重惩罚-50%)
│       └─ 疾行 (奔跑不额外消耗)
│           └─ 野外求生 (食物效果+50%)
│               └─ 危机直觉 (危险预警)

洞察系 (考古学家加成):
├─ 细节观察 (发现隐藏+20%)
│   └─ 模式识别 (识别迷宫规律)
│       └─ 第六感 (直觉提示)
│           └─ 古文字解读 (可解所有符号)
│               └─ 历史回溯 (看到过去影像)
```

### 4.2 技能学习

- 每升1级获得1技能点
- 技能需要前置技能
- 高级技能需要更高等级
- 可重置技能树（消耗金币）

## 5. 职业解锁

### 5.1 初始解锁

- 测绘师
- 探险家

### 5.2 条件解锁

| 职业 | 解锁条件 |
|------|----------|
| 考古学家 | 发现10个古代符号 |
| 生存专家 | 在压力>70时完成一层 |
| 信使 | 完成一次多人合作 |

## 6. 技术实现

### 6.1 数据结构

```gdscript
## 职业类型
enum ClassType {
    CARTOGRAPHER,   # 测绘师
    EXPLORER,       # 探险家
    ARCHAEOLOGIST,  # 考古学家
    SURVIVALIST,    # 生存专家
    MESSENGER,      # 信使
}

## 职业数据
class ClassData:
    var id: int
    var name: String
    var description: String
    var passive_bonuses: Dictionary
    var starting_equipment: Array
    var skill_affinity: String
    var difficulty: int

## 玩家职业状态
class PlayerClass:
    var class_type: int
    var skill_points: int = 0
    var unlocked_skills: Array = []
```

### 6.2 职业效果应用

```gdscript
## 应用职业被动
func apply_class_passives(player: Player, class_type: int) -> void:
    match class_type:
        ClassType.CARTOGRAPHER:
            player.drawing_accuracy += 0.3
            player.memory_decay_rate *= 0.8
        ClassType.EXPLORER:
            player.base_vision += 2
            player.move_speed *= 1.2
            player.stress_from_lost *= 0.5
        ClassType.SURVIVALIST:
            player.stamina_consumption *= 0.8
            player.stress_rate *= 0.7
            player.food_effect *= 1.5
```
