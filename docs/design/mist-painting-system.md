# 迷雾绘制系统

> **系统:** Mist Painting System
> **重要性:** ⭐ 核心机制
> **设计理念:** 玩家不是"看到"地图，而是"绘制"地图

---

## 1. 系统概述

### 1.1 核心概念

迷雾绘制是游戏的核心机制。玩家通过消耗墨水，在迷雾笼罩的世界中绘制可见区域。

### 1.2 三层视野模型

```
[真实视野]  ← 玩家实际能看到的范围（3-5格）
    ↓
[记忆地图]  ← 玩家绘制在纸上的地图（可能有误差）
    ↓
[迷雾区域]  ← 未探索或已遗忘的区域
```

## 2. 视野系统

### 2.1 真实视野 (True Sight)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 基础视野半径 | 3格 | 不受任何影响时的视野 |
| 火把照明 | +2格 | 手持火把时的加成 |
| 提灯照明 | +3格 | 更稳定的光源 |
| 黑暗环境 | -1格 | 无光源时的惩罚 |
| 迷雾区域 | -1格 | 特殊区域的惩罚 |

### 2.2 视野边缘效果

视野边缘不是"硬切"，而是渐进模糊：

```
  清晰    轻微模糊    重度模糊    不可见
   ●  →    ◐    →    ◑    →    ○
  100%     70%        30%        0%
```

### 2.3 光源影响

| 光源类型 | 视野加成 | 持续时间 | 特点 |
|----------|----------|----------|------|
| 火把 | +2格 | 10分钟 | 轻微闪烁 |
| 提灯 | +3格 | 30分钟 | 稳定 |
| 荧光石 | +1格 | 无限 | 微弱 |
| 魔法光球 | +4格 | 5分钟 | 可跟随 |

```yaml
  - 永久记录，无法修改
  - 精度高 (误差率: 5%)
```

```yaml
  - 极高精度 (误差率: 1%)
```

```yaml
获取: 隐藏宝箱，特殊事件
```

## 4. 地图精度与误差机制

### 4.1 误差产生原因

```
误差率 = 基础误差 + 环境惩罚 + 移动惩罚 + 工具修正
```

| 因素 | 误差加成 | 说明 |
|------|----------|------|
| 基础误差 | +10% | 任何绘制都有基础误差 |
| 移动中绘制 | +15% | 边走边画 |
| 黑暗环境 | +20% | 光源不足 |
| 迷雾区域 | +25% | 感知扭曲 |
| 魔法干扰 | +30% | 特殊区域 |
| 疲劳状态 | +10% | 体力低于30% |

### 4.2 误差表现形式

#### 距离误差
```
实际:    绘制结果:
┌───┐    ┌───┐
│ A ═══ B    │ A ═ B  (距离缩短)
└───┘    └───┘
```

#### 角度误差
```
实际:    绘制结果:
    B         B
   /          |
  /           |
 A            A─── (角度变直)
```

#### 连通性误差
```
实际:    绘制结果:
A ═══ B    A ═══ B
║     ║    ║     ║
C ═══ D    C     D (遗漏通道)
```

#### 幻觉误差（深层迷宫特有）
```
实际:    绘制结果:
┌───┐    ┌───┬───┐
│   │    │   │幻觉│
│ A │    │ A │通道│
└───┘    └───┴───┘
        (绘制了不存在的通道)
```

### 4.3 误差检测与修正

玩家可以通过以下方式发现误差：
- **回访验证** - 回到已绘制区域，发现不一致
- **测绘仪校准** - 使用测绘仪重新测量
- **古代地图碎片** - 对照真实地图
- **地标确认** - 特殊地标不会出错

## 5. 墨水系统

### 5.1 墨水属性

| 属性 | 默认值 | 说明 |
|------|--------|------|
| 初始墨水 | 50 | 游戏开始时的墨水量 |
| 最大墨水 | 100 | 墨水上限 |
| 恢复方式 | 收集/商店 | 获取途径 |

### 5.2 墨水消耗

| 操作 | 墨水消耗 | 说明 |
|------|----------|------|
| 铅笔绘制 | 0.5/次 | 低消耗 |
| 羽毛笔绘制 | 1.0/次 | 中消耗 |
| 测绘仪绘制 | 2.0/次 | 高消耗 |
| 魔法卷轴 | 100% | 一次性消耗全部 |
| 绘制标记 | 0.2/次 | 放置地图标记 |

### 5.3 墨水不足

- 墨水不足时无法绘制
- 墨水耗尽时自动停止绘制
- 可通过收集/商店补充

## 6. 记忆衰退系统

### 6.1 衰退机制

```
记忆清晰度 = 100% - (经过时间 / 衰退周期) × 衰退率
```

| 层级 | 衰退周期 | 衰退率 | 说明 |
|------|----------|--------|------|
| 表层 | 30分钟 | 20% | 记忆相对稳定 |
| 中层 | 20分钟 | 35% | 开始明显模糊 |
| 深层 | 10分钟 | 50% | 快速遗忘 |
| 混沌层 | 5分钟 | 80% | 几乎无法记住 |

### 6.2 衰退表现

```
清晰 (100-80%)    轻微模糊 (80-50%)    重度模糊 (50-20%)    遗忘 (20-0%)
┌───┐             ┌───┐                ┌───┐                ???
│   │             │   │                │···│
│ A │             │ ░ │                │░░░│
│   │             │   │                │···│
└───┘             └───┘                └───┘
线条清晰          线条变淡/抖动         只剩轮廓             完全空白
```

### 6.3 阻止衰退的方法

1. **定期回访** - 最简单的办法
2. **墨水绘制** - 羽毛笔绘制的地图不衰减
3. **记忆强化技能** - 减缓衰退速度
4. **锚定地标** - 重要地标记忆不会衰退
5. **魔法卷轴** - 永久记录

## 9. 技术实现

### 9.1 核心数据结构

```gdscript
## 迷雾绘制系统数据
class_name MistPaintingData

var mist_image: Image          # 迷雾图像
var mist_texture: ImageTexture # 迷雾纹理
var brush_size: float = 20.0   # 笔刷大小
var is_drawing: bool = false   # 是否正在绘制

## 覆盖率
var mist_coverage: float = 1.0 # 迷雾覆盖率
var cleared_pixels: int = 0    # 已清除像素
var total_pixels: int = 0      # 总像素
```

### 9.2 绘制流程

```
1. 玩家按下绘制键
2. 检查墨水是否足够
3. 获取鼠标世界位置
4. 在迷雾图像上绘制
5. 消耗墨水
6. 更新纹理
7. 计算覆盖率
```
