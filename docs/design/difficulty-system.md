# 难度系统

> **系统:** Difficulty System
> **重要性:** ⭐⭐ 核心系统
> **设计理念:** 让每个玩家都能找到适合自己的挑战

---

## 1. 系统概述

### 1.1 难度等级

游戏提供5个难度等级，从简单到混沌，满足不同玩家需求。

### 1.2 难度影响因素

- 敌人强度
- 资源稀缺度
- 记忆衰退速度
- 动态迷宫频率
- 压力积累速度

---

## 2. 难度等级详情

### 2.1 简单模式

```yaml
目标玩家: 新手、休闲玩家

敌人强度: -30%
资源获取: +30%
记忆衰退: -30%
动态迷宫: 关闭
压力积累: -20%

特点:
  - 详细教程提示
  - 更多恢复道具
  - 记忆更持久
  - 迷宫不变化

推荐: 首次游玩玩家
```

### 2.2 普通模式

```yaml
目标玩家: 核心玩家

敌人强度: 基准
资源获取: 基准
记忆衰退: 基准
动态迷宫: L3+开启
压力积累: 基准

特点:
  - 标准游戏体验
  - 平衡的资源管理
  - 深层迷宫开始变化

推荐: 大多数玩家
```

### 2.3 困难模式

```yaml
目标玩家: 挑战型玩家

敌人强度: +30%
资源获取: -30%
记忆衰退: +30%
动态迷宫: L2+开启
压力积累: +30%

特点:
  - 敌人更具威胁
  - 资源更加稀缺
  - 更快的记忆衰退
  - 更早的迷宫变化

推荐: 有经验的玩家
```

### 2.4 噩梦模式

```yaml
目标玩家: 硬核玩家

敌人强度: +50%
资源获取: -50%
记忆衰退: +50%
动态迷宫: L1+开启
压力积累: +50%

特点:
  - 极具挑战
  - 严重资源短缺
  - 快速遗忘
  - 从一开始迷宫就变化

推荐: 精通游戏的玩家
解锁: 通关普通模式
```

### 2.5 混沌模式

```yaml
目标玩家: 极限挑战者

敌人强度: +100%
资源获取: -70%
记忆衰退: +100%
动态迷宫: 全部，变化更快
压力积累: +100%

特点:
  - 极端挑战
  - 几乎无资源
  - 记忆快速消失
  - 迷宫持续剧烈变化

推荐: 最勇敢的玩家
解锁: 通关困难模式
```

## 4. 动态难度调整

### 4.1 自适应难度

系统会根据玩家表现微调难度：

| 玩家表现 | 系统调整 |
|----------|----------|
| 连续死亡3次 | 提示降低难度，轻微增强恢复道具 |
| 连续胜利5次 | 增加精英敌人出现概率 |
| 地图准确度>95% | 增加隐藏要素 |
| 长时间未遇敌 | 触发追逐事件 |

### 4.2 调整范围

- 最大调整幅度: ±10%
- 不改变已设定的难度等级
- 仅影响局内数值

## 5. Permadeath选项

### 5.1 死亡模式

| 模式 | 死亡后果 |
|------|----------|
| 标准 | 损失50%金币，回到最近检查点 |
| 硬核 | 损失所有物品，地图保留但模糊 |
| Permadeath | 角色重置，仅保留解锁内容 |

### 5.2 模式选择

玩家可以在游戏开始时选择死亡模式：
- 简单/普通: 只能选标准
- 困难: 可选标准或硬核
- 噩梦: 可选硬核或Permadeath
- 混沌: 只能选Permadeath

## 7. 技术实现

### 7.1 数据结构

```gdscript
## 难度类型
enum DifficultyLevel {
    EASY,       # 简单
    NORMAL,     # 普通
    HARD,       # 困难
    NIGHTMARE,  # 噩梦
    CHAOS,      # 混沌
}

## 难度配置
class DifficultyConfig:
    var enemy_strength: float = 1.0
    var resource_rate: float = 1.0
    var memory_decay: float = 1.0
    var stress_rate: float = 1.0
    var dynamic_maze_start_layer: int = 3
    var hallucination_multiplier: float = 1.0
```

### 7.2 难度应用

```gdscript
## 获取难度配置
func get_difficulty_config(level: int) -> DifficultyConfig:
    var config = DifficultyConfig.new()

    match level:
        DifficultyLevel.EASY:
            config.enemy_strength = 0.7
            config.resource_rate = 1.3
            config.memory_decay = 0.7
            config.dynamic_maze_start_layer = 999  # 关闭

        DifficultyLevel.HARD:
            config.enemy_strength = 1.3
            config.resource_rate = 0.7
            config.memory_decay = 1.3
            config.dynamic_maze_start_layer = 2

        DifficultyLevel.CHAOS:
            config.enemy_strength = 2.0
            config.resource_rate = 0.3
            config.memory_decay = 2.0
            config.dynamic_maze_start_layer = 0

    return config
```
