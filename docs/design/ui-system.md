# 迷雾绘者 UI系统设计文档

## 1. 概述

### 1.1 设计目标
- 提供直观、美观的用户界面
- 支持键盘和手柄导航
- 支持多语言本地化
- 支持主题换肤
- 响应式布局适配不同分辨率

### 1.2 设计原则
- **简洁性**: 界面元素清晰明了，避免视觉混乱
- **一致性**: 保持风格、交互、动画的一致性
- **反馈性**: 及时响应用户操作，提供视觉反馈
- **可访问性**: 支持键盘、手柄、触摸等多种输入方式

## 2. UI架构设计

### 2.1 架构模式: 组件化 + 状态管理

```
UI Layer
├── Screens (Views) - 屏幕层
├── Components (Widgets) - 组件层
└── Animations (Transitions) - 动画层

UIManager - UI管理器核心
├── NavigationManager - 导航管理
├── ThemeManager - 主题管理
└── LocalizationManager - 本地化

底层依赖
├── GameStateManager - 游戏状态
├── ConfigManager - 配置管理
└── EventBus - 事件总线
```

### 2.2 核心模块职责

| 模块 | 职责 |
|------|------|
| UIManager | UI系统的中央控制器，管理所有UI屏幕和组件 |
| ThemeManager | 管理UI主题、样式、颜色方案 |
| LocalizationManager | 管理多语言文本和字体 |
| NavigationManager | 处理焦点导航、输入映射 |
| AnimationManager | 管理UI动画和过渡效果 |

## 6. 动画系统

### 6.1 过渡动画
| 动画 | 时长 | 缓动函数 |
|------|------|----------|
| 淡入 | 0.3s | EASE_OUT |
| 淡出 | 0.2s | EASE_IN |
| 滑动 | 0.3s | EASE_OUT |
| 缩放 | 0.2s | EASE_OUT_BACK |

### 6.2 交互动画
| 动画 | 时长 | 说明 |
|------|------|------|
| 按钮悬停 | 0.15s | 放大、变色 |

| 焦点切换 | 0.2s | 边框高亮 |
| 数值变化 | 0.3s | 数字滚动 |

## 8. 输入系统

### 8.1 导航映射
| 操作 | 键盘 | 手柄 |
|------|------|------|
| 向上导航 | W / ↑ | 左摇杆上 / DPad上 |
| 向下导航 | S / ↓ | 左摇杆下 / DPad下 |
| 向左导航 | A / ← | 左摇杆左 / DPad左 |
| 向右导航 | D / → | 左摇杆右 / DPad右 |
| 确认 | Enter / Space | A键 |
| 取消 / 返回 | Esc / Backspace | B键 |
| 暂停 | Esc | Start键 |

### 8.2 焦点管理
- 自动焦点：打开菜单时自动聚焦第一个可交互元素
- 记忆焦点：返回上级菜单时恢复之前的焦点位置
- 循环导航：到达边界时循环到另一端
