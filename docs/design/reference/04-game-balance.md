# 《迷雾绘者》- 数值平衡系统

> **系统:** Game Balance  
> **设计理念:** 数值是游戏的手感。太简单则无聊，太难则挫败。

---

## 1. 体力系统

### 1.1 体力消耗

| 动作 | 基础消耗 | 影响因素 |
|------|----------|----------|
| 移动1格 | 1点 | 负重+50%，疲劳+100% |
| 奔跑1格 | 2点 | 速度翻倍 |
| 绘制地图 | 5点 | 工具类型影响时间 |
| 仔细测绘 | 10点 | 精度提升 |
| 搜索/检查 | 8点 | 发现隐藏要素 |
| 战斗 | 15-30点 | 根据敌人强度 |
| 休息恢复 | -20点/分钟 | 需要安全区域 |

### 1.2 体力上限

```
基础上限: 100点
每升1级: +10点
最大上限: 200点
```

### 1.3 疲劳状态

| 体力百分比 | 状态 | 效果 |
|------------|------|------|
| 100-70% | 精力充沛 | 无惩罚 |
| 70-40% | 正常 | 无惩罚 |
| 40-20% | 疲劳 | 移动速度-10%，绘制误差+10% |
| 20-10% | 精疲力竭 | 移动速度-25%，绘制误差+25%，视野-1 |
| 10-0% | 濒死 | 移动速度-50%，无法绘制，视野-2 |
| 0% | 昏迷 | 需要救援或等待恢复 |

### 1.4 恢复方式

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 自然恢复 | 5点/分钟 | 安全区域，不移动 |
| 休息 | 20点/分钟 | 坐下休息，易受攻击 |
| 食物 | 10-50点 | 消耗品 |
| 药水 | 立即50点 | 稀有，有副作用 |
| 睡眠 | 100% | 只能在特定房间 |

## 3. 工具耐久系统

### 3.1 耐久消耗

| 工具 | 耐久总量 | 每次使用消耗 | 预计使用次数 |
|------|----------|--------------|--------------|
| 铅笔 | 100 | 1 | 100次绘制 |
| 羽毛笔 | 50 | 1 | 50次绘制 |
| 墨水 | 50 | 1 | 50次（与羽毛笔绑定） |
| 测绘仪 | 无限 | 0 | 无限 |
| 橡皮 | 30 | 1 | 30次修改 |

### 3.2 纸张限制

| 类型 | 尺寸 | 背包堆叠 | 特性 |
|------|------|----------|------|
| 粗糙纸 | 20×20 | 5张 | 遇水即毁 |
| 羊皮纸 | 30×30 | 5张 | 标准 |
| 防水纸 | 25×25 | 3张 | 防水 |
| 魔法纸 | 40×40 | 2张 | 永不破损 |

### 3.3 修复与补充

```
铅笔: 无法修复，需要携带多支或找到削笔工具(+20耐久)
羽毛笔: 无法修复，但可找到替换
墨水: 可补充，找到墨水瓶(+30)
测绘仪: 精密仪器，损坏后需要专业修理
```

## 4. 背包系统

### 4.1 容量管理

```yaml
基础背包: 10格

物品占用:
  地图纸张: 1格 (可堆叠5张同类型)
  绘制工具: 1-2格
  光源: 1格 (火把可堆叠3个，提灯占2格)
  食物: 1格 (可堆叠10个)
  药水: 1格 (可堆叠5个)
  材料: 1格 (可堆叠20个)
  任务物品: 1-3格 (通常不可堆叠)
  
扩展:
  背包升级I: +3格
  背包升级II: +5格
  魔法袋: 纸张类不占格
```

### 4.2 负重影响

| 负重比例 | 效果 |
|----------|------|
| 0-50% | 无影响 |
| 50-75% | 移动速度-10%，体力消耗+20% |
| 75-90% | 移动速度-25%，体力消耗+50%，无法奔跑 |
| 90-100% | 移动速度-50%，体力消耗+100%，无法绘制 |
| >100% | 无法移动 |

## 7. 难度调节

### 7.1 难度等级

| 难度 | 敌人强度 | 资源稀缺 | 记忆衰退 | 动态迷宫 |
|------|----------|----------|----------|----------|
| 简单 | -30% | +30% | -30% | 关闭 |
| 普通 | 基准 | 基准 | 基准 | L3+ |
| 困难 | +30% | -30% | +30% | L2+ |
| 噩梦 | +50% | -50% | +50% | L1+ |
| 混沌 | +100% | -70% | +100% | 全部，变化更快 |

### 7.2 Permadeath选项

```
标准模式: 死亡损失50%金币，回到入口
硬核模式: 死亡损失所有物品，地图保留但模糊
Permadeath: 死亡后角色重置，仅保留解锁内容
```

## 9. 心理状态系统 (v1.1)

### 9.1 压力值机制

```typescript
interface MentalState {
  stressLevel: number;      // 0-100
  hallucinationChance: number; // 基于压力的概率
  drawingAccuracyPenalty: number; // 压力对绘图的影响
}
```

**压力来源:**

| 情况 | 压力增加 | 说明 |
|------|----------|------|
| 在黑暗中每5分钟 | +5 | 未知带来恐惧 |
| 迷路时每分钟 | +10 | 恐慌累积 |
| 发现尸体/遗迹 | +15 | 心理冲击 |
| 被追逐时 | +20 | 生存压力 |
| 地图出现矛盾 | +8 | 认知失调 |
| 体力低于20% | +10 | 生理影响心理 |

**压力影响:**

| 压力范围 | 状态 | 效果 |
|----------|------|------|
| 0-30 | 冷静 | 无惩罚，可能+5%精度 |
| 30-50 | 紧张 | 绘图误差+10% |
| 50-70 | 焦虑 | 误差+20%，手抖动画 |
| 70-85 | 恐慌 | 误差+35%，开始出现幻觉 |
| 85-100 | 崩溃 | 误差+50%，频繁幻觉，视野闪烁 |

### 9.2 幻觉系统

**幻觉类型:**

1. **虚假通道** - 在墙上显示不存在的门
2. **错误标记** - 已探索区域显示为未探索
3. **幽灵敌人** - 看到不存在的怪物
4. **地图扭曲** - 已绘制的线条轻微偏移
5. **声音幻听** - 听到脚步声、低语

**幻觉检测:**
- 使用测绘仪可以验证
- 与其他玩家地图对比
- 高压力消退后自动发现

### 9.3 压力恢复

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 安全区域休息 | -15/分钟 | 无威胁 |
| 完成准确绘图 | -20 | 准确度>90% |
| 镇定药水 | -30 | 消耗品 |
| 烟草/草药 | -10 | 轻微副作用 |
| 发现出口 | -40 | 希望恢复 |

## 10. 职业/专长系统 (v1.2)

### 10.1 探险者类型

| 职业 | 特性 | 起始装备 |
|------|------|----------|
| **测绘师** | 绘图精度+30%，更多纸张 | 测绘仪+羊皮纸×5 |
| **探险家** | 视野+2格，移动速度+20% | 提灯+火把×3 |
| **考古学家** | 解读符号，发现隐藏门×2 | 古文字手册 |
| **生存专家** | 体力消耗-20%，压力积累-30% | 食物×5+药水×2 |
| **信使** | 地图副本×2，适合合作 | 魔法纸×3 |

### 10.2 技能树

```
制图系:
├─ 精准测绘 (误差-10%)
│   └─ 快速绘图 (速度+30%)
│       └─ 记忆强化 (衰退-20%)
└─ 多图层绘制 (可同时看多层)
    └─ 立体感知 (预判结构)
        └─ 预知路径 (短暂显示出口方向)

生存系:
├─ 体能训练 (体力上限+20)
│   └─ 轻装上阵 (负重惩罚-50%)
│       └─ 疾行 (奔跑不额外消耗)
└─ 野外求生 (食物效果+50%)
    └─ 资源回收 (拾取更多)
        └─ 危机直觉 (危险预警)

洞察系:
├─ 细节观察 (发现隐藏+20%)
│   └─ 模式识别 (识别迷宫规律)
│       └─ 第六感 (直觉提示)
└─ 古文字解读 (可解所有符号)
    └─ 遗迹共鸣 (与古代地图互动)
        └─ 历史回溯 (看到过去影像)
```

## 11. 死亡与传承系统 (v1.2)

### 11.1 死亡惩罚

- 丢失当前携带物品
- 地图知识保留但精度-30%
- 可留下遗言标记

### 11.2 传承机制

- 新角色继承前角色20%技能经验
- 可找到前角色尸体和遗物
- 回收前角色地图（最宝贵）

### 11.3 幽灵模式

死亡后短暂幽灵形态:
- 可观察周围但不能互动
- 可留下标记警告后来者
- 帮助下一个自己
