# 迷雾绘者 - 核心玩法设计文档

> **版本:** v2.0
> **核心概念:** 探索 → 记录 → 收集 → 撤离

---

## 1. 核心玩法概述

### 1.1 一句话描述
> 进入未知迷宫，用手中的记录道具绘制地图，收集财宝后安全撤离。

### 1.2 核心循环
```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  进入    │ → │  探索    │ → │  记录    │ → │  撤离    │
│  迷宫    │    │  收集    │    │  地图    │    │  返回    │
└──────────┘    └──────────┘    └──────────┘    └────┬─────┘
     ↑                                               │
     └───────────────────────────────────────────────┘
```

### 1.3 游戏特色
- **手动记录地图** - 不是自动全开，玩家必须主动使用记录道具
- **资源有限** - 记录道具、体力、背包空间都是有限资源
- **风险与回报** - 探索越深，财宝越多，但风险越大
- **死亡惩罚** - 死亡丢失背包所有物资，高风险高回报

## 2. 记录系统（核心机制）

### 2.1 记录道具
```yaml
名称: 地图纸张/记录卷轴
功能: 记录玩家当前位置为中心的 3x3 区域
消耗: 一次性使用，用完即毁
获取方式:
  - 进入迷宫前购买
  - 迷宫中拾取
携带上限: 20个（由背包容量决定）
```

### 2.2 地图道具
```yaml
名称: 探险地图
功能: 显示玩家已记录的所有区域
类型: 道具，使用时打开查看
特点:
  - 只显示已记录的区域
  - 记录永久保存，不会消失
  - 死亡后地图保留（但丢失其他物资）
```

### 2.3 记录操作流程
```
1. 玩家在迷宫中发现新区域
2. 决定是否使用记录道具
3. 选择使用道具 → 记录玩家为中心 3x3 区域
4. 该区域永久添加到地图
5. 道具消耗 -1
```

## 3. 迷宫系统

### 3.1 迷宫结构
- **入口/出口**: 同一个位置，玩家从这里进入和撤离
- **楼层系统**: 通过楼梯上下层，每层大小由迷宫生成器决定
- **撤离道具**: 特殊道具，使用后直接返回入口

### 3.2 迷宫内容

#### 财宝（收集目标）
| 类型 | 价值 | 稀有度 |
|------|------|--------|
| 金币 | 低 | 常见 |
| 宝石 | 中 | 较少 |
| 文物 | 高 | 稀有 |
| 传说物品 | 极高 | 极稀有 |

#### 敌人（威胁）
- 不同种类敌人，也是探索方向之一
- 敌人分布随机
- 战斗系统（待设计）

#### 陷阱（危险）
- 触发后造成伤害或负面效果
- 可被发现和规避

### 3.3 楼层设计
| 层数 | 复杂度 | 敌人密度 | 财宝价值 | 备注 |
|------|--------|----------|----------|------|
| 1层 | 低 | 低 | 低 | 新手区域 |
| 2层 | 中 | 中 | 中 | 标准区域 |
| 3层+ | 高 | 高 | 高 | 深层危险区域 |

## 4. 资源系统

### 4.1 背包系统
```yaml
设计参考: 类似三角洲行动
容量上限: 开发初期固定20格
物品占用: 每个物品占用1格
死亡惩罚: 丢失背包所有物资
```

### 4.2 体力系统
```yaml
特点: 不自动回复
消耗:
  - 移动消耗体力
  - 战斗消耗体力
  - 特殊动作消耗体力
恢复: 需要通过道具或食物补充
归零: 体力归零后无法行动，可能死亡
```

### 4.3 光源系统
- 仅特定迷宫需要（如深层迷宫、黑暗区域）
- 光源影响视野范围
- 光源有持续时间限制

## 5. 游戏流程

### 5.1 准备阶段
```
1. 查看当前资源（金币、记录道具、装备）
2. 购买/补充物资
3. 选择要探索的迷宫
4. 进入迷宫
```

### 5.2 探索阶段
```
1. 在迷宫中移动
2. 发现新区域
3. 决定是否记录（消耗道具）
4. 收集财宝
5. 应对敌人/陷阱
6. 管理体力
7. 决定继续深入或撤离
```

### 5.3 撤离阶段
```
方式一: 走回入口
方式二: 使用撤离道具直接返回

成功撤离: 保留所有收集的物资
死亡: 丢失背包所有物资
```

### 5.4 结算阶段
```
1. 清点收获（财宝价值）
2. 地图记录保存
3. 返回准备界面
```

## 6. 关键决策点

### 6.1 记录决策
- **记录还是不记录？** 记录道具有限
- **现在记录还是继续探索？** 可能找到更好的区域
- **记录哪里？** 选择有价值的位置记录

### 6.2 探索决策
- **继续深入还是撤离？** 深层有更好的财宝，但更危险
- **探索未知区域还是走已知路线？** 风险与回报

### 6.3 资源决策
- **体力不足时怎么办？** 使用道具还是返回？
- **背包快满时怎么办？** 丢弃低价值物品还是返回？
