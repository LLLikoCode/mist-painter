# 光源系统

> **系统:** Light System
> **重要性:** ⭐⭐ 核心机制
> **设计理念:** 光明带来希望，也消耗资源

---

## 1. 系统概述

### 1.1 核心概念

光源是迷宫探索的关键资源。不同的光源有不同的特性，玩家需要合理管理光源以保证视野。

### 1.2 视野与光源关系

```
基础视野 + 光源加成 = 总视野范围
```

### 4.2 燃料管理

```gdscript
## 光源燃料系统
class_name LightFuelSystem

var current_fuel: float = 0.0
var max_fuel: float = 100.0
var consumption_rate: float = 1.0  # 每分钟消耗

func update(delta: float) -> void:
    if is_active:
        current_fuel -= consumption_rate * delta / 60.0
        if current_fuel <= 0:
            current_fuel = 0
            deactivate()

func refuel(amount: float) -> void:
    current_fuel = min(max_fuel, current_fuel + amount)
```

## 5. 黑暗惩罚

### 5.1 黑暗等级

| 黑暗程度 | 视野惩罚 | 移动惩罚 | 绘制惩罚 |
|----------|----------|----------|----------|
| 微光 | -1格 | 无 | 误差+10% |
| 昏暗 | -2格 | 速度-10% | 误差+25% |
| 黑暗 | -3格 | 速度-25% | 无法绘制 |
| 漆黑 | 仅1格 | 速度-50% | 无法绘制 |

### 5.2 黑暗效果

```gdscript
## 计算视野
func calculate_vision(base_vision: int, light_source: LightSource) -> int:
    var total_vision = base_vision

    # 光源加成
    if light_source and light_source.is_active:
        total_vision += light_source.vision_bonus

    # 环境惩罚
    if is_in_dark_area():
        total_vision -= 1
    if is_in_mist_area():
        total_vision -= 1

    return max(1, total_vision)
```

## 6. 光源特效

### 6.1 火把效果

- 亮度随机波动 (0.9-1.1)
- 轻微位置抖动
- 火星粒子效果
- 燃烧音效

### 6.2 提灯效果

- 稳定的光照
- 可关闭节省燃料
- 玩家移动时轻微摇摆

### 6.3 魔法光球效果

- 跟随玩家移动
- 平滑的光照边缘
- 魔法粒子效果
