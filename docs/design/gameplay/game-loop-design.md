## 核心游戏循环概述

### 1.1 玩家主要活动流程

迷雾绘者的核心游戏循环遵循经典的"探索-战斗-成长"模式，融入独特的"迷雾驱散"机制：

### 1.2 单次游戏会话设计

**目标时长**: 15-30分钟

| 阶段 | 预计时长 | 占比 | 说明 |
|------|---------|------|------|
| 探索阶段 | 8-12分钟 | 40% | 迷宫移动、迷雾驱散、路径选择 |
| 战斗阶段 | 6-10分钟 | 35% | 遭遇战、Boss战 |
| 资源管理 | 2-4分钟 | 15% | 商店、道具使用、状态恢复 |
| 决策/升级 | 1-2分钟 | 10% | 技能选择、路线规划 |

**会话结构**:
```
[开始界面] → [第1层: 教学/简单] → [第2层: 进阶] → [第3层: Boss战] → [结算/结束]
    │              │                    │               │              │
  0:00           2:00                10:00           22:00          30:00
```

### 1.3 长期进度循环

**多周目系统**:
- **第1周目**: 基础体验，解锁基础技能
- **第2-3周目**: 解锁进阶内容，新敌人、新事件
- **第4周目+**: 无尽模式，挑战更高难度

**成就收集**:
- 探索类: "迷雾驱散者"(驱散100格迷雾)、"全图探索"(单局100%探索)
- 战斗类: "无伤通关"、"速通大师"(15分钟内完成)
- 收集类: "全图鉴"、"墨水富翁"(单局获得1000墨水)

## 探索阶段设计

### 2.1 迷宫生成机制

**迷宫规格**:
| 层级 | 网格大小 | 房间数 | 特殊房间 | 迷雾密度 |
|------|---------|--------|----------|----------|
| 第1层 | 8x8 | 12-15 | 1商店+2休息点 | 60% |
| 第2层 | 10x10 | 18-22 | 1商店+1Boss前休息点 | 70% |
| 第3层 | 12x12 | 25-30 | 1商店+最终Boss | 80% |

**生成算法**:
1. **随机漫步算法**: 生成基础路径连接
2. **房间放置**: 在路径节点放置房间
3. **迷雾覆盖**: 初始状态所有非起始区域被迷雾覆盖
4. **事件分布**: 根据权重随机分配事件类型

### 2.2 房间事件触发

**事件类型分布**:

| 事件类型 | 概率 | 说明 |
|----------|------|------|
| 普通战斗 | 35% | 标准敌人遭遇 |
| 精英战斗 | 15% | 强化敌人，更好奖励 |
| 宝箱 | 15% | 道具/装备/墨水奖励 |
| 商店 | 10% | 购买道具和升级 |
| 休息点 | 10% | 恢复HP/SP |
| 陷阱 | 8% | 受到伤害或负面效果 |
| 特殊事件 | 7% | 随机剧情/奖励事件 |

**事件触发规则**:
- 首次进入房间必定触发事件
- 已清理房间可安全通过
- Boss房间必须击败Boss才能离开

### 2.3 迷雾驱散与地图揭示

**迷雾机制**:
```
迷雾状态:
┌─────────────┬─────────────┬─────────────┐
│   完全迷雾   │   半透明    │   已揭示    │
│   [未知]    │   [感知]    │   [清晰]    │
├─────────────┼─────────────┼─────────────┤
│ 完全黑暗    │ 可见轮廓    │ 完全可见    │
│ 无法进入    │ 可识别类型  │ 可交互      │
│ 危险未知    │ 风险可评估  │ 信息完整    │
└─────────────┴─────────────┴─────────────┘
```

**驱散方式**:
1. **邻近驱散**: 进入房间后，相邻房间迷雾变为半透明
2. **技能驱散**: 使用"洞察"技能驱散大范围迷雾
3. **道具驱散**: 使用"照明弹"等道具揭示区域
4. **永久驱散**: 击败房间敌人后，该房间永久清晰

### 2.4 路径选择与风险回报

**路径决策要素**:
- **距离**: 到出口的最短路径 vs 探索更多房间
- **状态**: 当前HP/SP是否允许挑战精英敌人
- **资源**: 墨水数量是否足够在商店消费
- **迷雾**: 未知区域的风险评估

**风险回报设计**:
```
低风险路径 ──────────────────────────────▶ 高风险路径
    │                                          │
    ▼                                          ▼
 短距离、已知敌人                      长距离、未知区域
 基础奖励                              稀有道具、大量墨水
 安全但收益低                          危险但收益高
```

## 战斗阶段设计

### 3.1 遭遇触发机制

**触发条件**:
1. **主动遭遇**: 进入有敌人的房间
2. **陷阱遭遇**: 触发陷阱房间，强制战斗
3. **追击遭遇**: 从精英房间逃跑后，可能被追击

**敌人配置**:
| 房间类型 | 敌人数量 | 敌人类型 | 难度系数 |
|----------|----------|----------|----------|
| 普通房间 | 1-2 | 普通敌人 | 1.0x |
| 精英房间 | 1 | 精英敌人 | 1.5x |
| Boss房间 | 1 | Boss | 2.0-3.0x |

### 3.2 回合制战斗流程

**战斗循环**:

**玩家行动选项**:
1. **攻击**: 基础攻击，消耗少量SP
2. **技能**: 特殊能力，消耗SP，有冷却
3. **道具**: 使用背包中的消耗品
4. **逃跑**: 尝试逃离战斗（成功率取决于敌人类型）

### 3.3 战斗结束条件

**胜利条件**:
- 所有敌人HP归零
- 奖励: 经验值 + 墨水 + 可能掉落道具

**失败条件**:
- 玩家HP归零
- 结果: 游戏结束，保留部分永久进度

**逃跑条件**:
- 普通敌人: 75%成功率
- 精英敌人: 50%成功率
- Boss: 不可逃跑
- 逃跑惩罚: 不获得奖励，可能遭遇追击

### 3.4 战斗奖励分配

**基础奖励公式**:
```
经验值 = 敌人基础经验 × 难度系数 × (1 + 层数×0.2)
墨水 = 敌人基础墨水 × 难度系数 × (1 + 幸运加成)
```

**掉落系统**:
| 敌人类型 | 普通道具 | 稀有道具 | 装备掉落率 |
|----------|----------|----------|------------|
| 普通敌人 | 30% | 5% | 10% |
| 精英敌人 | 50% | 15% | 25% |
| Boss | 80% | 40% | 50% |

## 资源管理阶段

### 4.1 生命值/精神值管理

**HP (生命值)**:
- **初始值**: 100点
- **上限**: 随等级提升，最高200点
- **恢复方式**:
  - 休息点: 恢复50% HP
  - 道具: 药水恢复20-50点
  - 技能: 部分技能可恢复HP

**SP (精神值)**:
- **初始值**: 50点
- **上限**: 随等级提升，最高100点
- **用途**: 释放技能、使用特殊道具
- **恢复方式**:
  - 休息点: 完全恢复
  - 道具: 精神药水恢复20-30点
  - 战斗结束: 恢复10点

**状态监控**:
```
┌─────────────────────────────────────────────────────────┐
│ 状态条设计                                               │
├─────────────────────────────────────────────────────────┤
│ HP: [████████░░░░░░░░░░] 80/100  (绿色=健康,黄色=警告) │
│ SP: [██████░░░░░░░░░░░░] 30/50   (蓝色=充足,红色=危险) │
└─────────────────────────────────────────────────────────┘
```

### 4.2 道具使用决策

**道具分类**:

| 类型 | 道具示例 | 效果 | 使用时机 |
|------|----------|------|----------|
| 恢复类 | 生命药水 | +30 HP | 战斗外/战斗中 |
| 恢复类 | 精神药水 | +20 SP | 战斗外/战斗中 |
| 攻击类 | 炸弹 | 造成50伤害 | 战斗中 |
| 防御类 | 护盾卷轴 | 免疫下次伤害 | 战斗前 |
| 功能类 | 照明弹 | 揭示3x3区域 | 探索时 |
| 功能类 | 地图碎片 | 显示当前层地图 | 任意时刻 |

**背包限制**:
- **容量**: 20格（可扩展至30格）
- **堆叠**: 同类道具最多堆叠10个
- **管理策略**: 需要定期清理，做出取舍

### 4.3 装备升级时机

**装备系统**:

| 部位 | 装备类型 | 属性加成 |
|------|----------|----------|
| 武器 | 画笔/法杖 | 攻击力+5~+20 |
| 防具 | 斗篷/长袍 | 防御力+3~+10 |
| 饰品 | 戒指/项链 | 特殊效果 |

**升级时机建议**:
1. **Boss战前**: 确保装备达到当前层推荐等级
2. **商店刷新**: 遇到高品质装备时优先考虑
3. **属性瓶颈**: 当战斗变得困难时检查装备

### 4.4 货币（墨水）获取与消费

**墨水获取途径**:

| 来源 | 数量范围 | 备注 |
|------|----------|------|
| 战斗胜利 | 10-50 | 随难度增加 |
| 宝箱开启 | 20-100 | 随机数量 |
| 出售道具 | 道具价值的50% | 商店出售 |
| 特殊事件 | 50-200 | 随机事件奖励 |

**墨水消费项目**:

| 项目 | 价格范围 | 优先级 |
|------|----------|--------|
| 生命药水 | 20-30 | 高 |
| 精神药水 | 15-25 | 高 |
| 装备购买 | 100-500 | 中 |
| 技能升级 | 200-1000 | 中 |
| 地图服务 | 50 | 低 |
| 复活服务 | 500 | 紧急 |

**经济平衡**:
- 单局预期总收入: 800-1200墨水
- 必要支出: 400-600墨水
- 可支配: 200-600墨水（用于升级/装备）

## 节奏控制

### 6.1 紧张与放松的节奏交替

**节奏曲线设计**:
```
紧张度
  100% │                                    ╱╲
       │                              ╱╲  ╱  ╲
   75% │                        ╱╲  ╱  ╲╱    ╲
       │                  ╱╲  ╱  ╲╱            ╲
   50% │            ╱╲  ╱  ╲╱                    ╲
       │      ╱╲  ╱  ╲╱                            ╲
   25% │╱╲  ╱  ╲╱                                    ╲
       │  ╲╱                                            ╲
    0% └──────────────────────────────────────────────────▶ 时间
       开始   第1层      第2层      第3层      Boss      结束
```

**节奏控制要素**:
| 阶段 | 紧张度 | 设计要点 |
|------|--------|----------|
| 开始 | 低 | 教学引导，轻松探索 |
| 第1层 | 中低 | 建立信心，适度挑战 |
| 第2层 | 中 | 增加难度，引入精英 |
| 第3层 | 中高 | 资源紧张，需要策略 |
| Boss战 | 高 | 紧张刺激，技能全开 |
| 结束 | 低 | 结算奖励，成就感 |

### 6.2 难度曲线与玩家技能匹配

**动态难度调整**:

| 玩家表现 | 调整措施 |
|----------|----------|
| 连续胜利 | 增加敌人强度+10% |
| 连续失败 | 降低敌人强度-10%，增加恢复道具 |
| 完美战斗 | 下一战精英概率+20% |
| 濒死胜利 | 自动触发恢复事件 |

**学习曲线**:
- **第1层**: 基础操作教学，简单敌人
- **第2层**: 引入机制组合，需要策略
- **第3层**: 综合运用，高难度挑战

### 6.3 意外事件与惊喜设计

**随机事件表**:

| 事件名称 | 触发概率 | 效果 | 玩家选择 |
|----------|----------|------|----------|
| 神秘商人 | 5% | 出售稀有物品 | 购买/离开 |
| 隐藏房间 | 3% | 额外宝箱 | 探索/忽略 |
| 迷雾异变 | 2% | 敌人强化但奖励翻倍 | 挑战/回避 |
| 古老祭坛 | 2% | 献祭HP换取SP | 献祭/离开 |
| 迷路旅人 | 3% | 给予道具或战斗 | 帮助/攻击 |
| 时空裂隙 | 1% | 传送至随机位置 | 进入/回避 |

**惊喜元素**:
- 隐藏Boss: 满足条件后出现的强力敌人
- 彩蛋房间: 致敬经典游戏的特殊房间
- 连击奖励: 连续快速击败敌人的额外奖励

**敌人数值（第1层）**:

|----------|-----|--------|--------|----------|----------|

| 普通敌人B | 40 | 6 | 4 | 25 | 20 |
| 精英敌人 | 80 | 15 | 8 | 60 | 50 |
| Boss | 200 | 20 | 12 | 300 | 200 |

### 8.2 战斗平衡公式

**伤害计算公式**:
```
基础伤害 = 攻击力 × 技能倍率 - 防御力
实际伤害 = 基础伤害 × 随机系数(0.9-1.1) × 暴击倍率(1.5x)
```

**战斗回合预期**:
- 普通战斗: 3-5回合
- 精英战斗: 5-8回合
- Boss战: 10-15回合

**资源消耗预期**:
- 普通战斗: 5-10 SP
- 精英战斗: 15-25 SP
- Boss战: 40-60 SP

### 8.3 经济平衡建议

**收入与支出比例**:
```
单局预期:
├── 总收入: 1000墨水
│   ├── 战斗: 600 (60%)
│   ├── 宝箱: 300 (30%)
│   └── 事件: 100 (10%)
│
└── 总支出: 800墨水
    ├── 必要消耗: 500 (62.5%)
    │   ├── 药水: 300
    │   └── 装备修理: 200
    └── 可选消费: 300 (37.5%)
        ├── 装备升级: 200
        └── 技能学习: 100

净收益: 200墨水 (用于永久升级)
```

### 8.4 进度节奏建议

**单次游戏进度**:
- 等级提升: 预期2-3级/局
- 技能解锁: 每局解锁1-2个新技能
- 装备更新: 每局获得2-3件新装备

**多周目成长**:
- 每完成一周目，基础属性+5%
- 解锁新技能/装备池
- 敌人强度同步提升，保持挑战
