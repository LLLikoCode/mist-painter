# 地图绘制系统 - 核心机制设计

> 这是游戏的核心差异化机制。玩家不是"看到"地图，而是"绘制"地图。

---

## 1. 视野与迷雾系统

### 三层视野模型

```
[真实视野]  ← 玩家实际能看到的范围（3-5格）
    ↓
[记忆地图]  ← 玩家绘制在纸上的地图（可能有误差）
    ↓
[迷雾区域]  ← 未探索或已遗忘的区域
```

### 真实视野 (True Sight)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 基础视野半径 | 3格 | 不受任何影响时的视野 |
| 火把照明 | +2格 | 手持火把时的加成 |
| 提灯照明 | +3格 | 更稳定的光源 |
| 黑暗环境 | -1格 | 无光源时的惩罚 |
| 迷雾区域 | -1格 | 特殊区域的惩罚 |

### 视野边缘效果

视野边缘不是"硬切"，而是渐进模糊：

```
  清晰    轻微模糊    重度模糊    不可见
   ●  →    ◐    →    ◑    →    ○
  100%     70%        30%        0%
```

## 2. 绘制工具系统

### 工具类型详解

#### 2.1 铅笔 (Pencil)
```yaml
名称: 探险者铅笔
耐久: 100笔划
特性:
  - 可擦除修改
  - 绘制速度快
  - 精度中等 (误差率: 15%)
缺点:
  - 容易磨损
  - 遇水模糊（潮湿区域风险）
获取: 初始装备，商店常见
```

#### 2.2 羽毛笔+墨水 (Quill & Ink)
```yaml
名称: 学者羽毛笔
耐久: 50笔划（墨水限制）
特性:
  - 永久记录，无法修改
  - 精度高 (误差率: 5%)
  - 可绘制细节标记
缺点:
  - 墨水有限
  - 错误无法修正
获取: 中层迷宫宝箱，商店稀有
```

#### 2.3 测绘仪 (Surveying Tool)
```yaml
名称: 精密测绘仪
耐久: 无限（但使用耗时）
特性:
  - 极高精度 (误差率: 1%)
  - 可测量距离和角度
  - 发现隐藏结构
缺点:
  - 占用2格背包
  - 使用时不能移动
  - 绘制速度极慢
获取: 深层迷宫，任务奖励
```

#### 2.4 魔法卷轴 (Magic Scroll)
```yaml
名称: 记忆卷轴
耐久: 一次性
特性:
  - 自动记录当前区域
  - 100%准确
  - 永久不衰退
缺点:
  - 极其稀有
  - 范围有限（当前房间/区域）
获取: 隐藏宝箱，特殊事件
```

### 工具对比表

| 工具 | 精度 | 速度 | 可修改 | 耐久 | 背包占用 |
|------|------|------|--------|------|----------|
| 铅笔 | ★★★ | ★★★★★ | ✓ | 100划 | 1格 |
| 羽毛笔 | ★★★★ | ★★★ | ✗ | 50划 | 1格 |
| 测绘仪 | ★★★★★ | ★ | - | 无限 | 2格 |
| 魔法卷轴 | ★★★★★ | ★★★★★ | - | 1次 | 1格 |

## 3. 地图精度与误差机制

### 误差产生原因

```
误差率 = 基础误差 + 环境惩罚 + 移动惩罚 + 工具修正
```

| 因素 | 误差加成 | 说明 |
|------|----------|------|
| 基础误差 | +10% | 任何绘制都有基础误差 |
| 移动中绘制 | +15% | 边走边画 |
| 黑暗环境 | +20% | 光源不足 |
| 迷雾区域 | +25% | 感知扭曲 |
| 魔法干扰 | +30% | 特殊区域 |
| 疲劳状态 | +10% | 体力低于30% |

### 误差表现形式

误差不是简单的"错误"，而是有多种表现：

#### 3.1 距离误差
```
实际:    绘制结果:
┌───┐    ┌───┐
│   │    │   │
│ A ═══ B    │ A ═ B  (距离缩短)
│   │    │   │
└───┘    └───┘
```

#### 3.2 角度误差
```
实际:    绘制结果:
    B         B
   /          |
  /           |
 A            A─── (角度变直)
```

#### 3.3 连通性误差
```
实际:    绘制结果:
A ═══ B    A ═══ B
║     ║    ║     ║
C ═══ D    C     D (遗漏通道)
```

#### 3.4 幻觉误差（深层迷宫特有）
```
实际:    绘制结果:
┌───┐    ┌───┬───┐
│   │    │   │幻觉│
│ A │    │ A │通道│
│   │    │   │    │
└───┘    └───┴───┘
        (绘制了不存在的通道)
```

### 误差检测与修正

玩家可以通过以下方式发现误差：
- **回访验证** - 回到已绘制区域，发现不一致
- **测绘仪校准** - 使用测绘仪重新测量
- **古代地图碎片** - 对照真实地图
- **地标确认** - 特殊地标不会出错

## 4. 记忆衰退系统

### 衰退机制

```
记忆清晰度 = 100% - (经过时间 / 衰退周期) × 衰退率
```

| 层级 | 衰退周期 | 衰退率 | 说明 |
|------|----------|--------|------|
| 表层 | 30分钟 | 20% | 记忆相对稳定 |
| 中层 | 20分钟 | 35% | 开始明显模糊 |
| 深层 | 10分钟 | 50% | 快速遗忘 |
| 混沌层 | 5分钟 | 80% | 几乎无法记住 |

### 衰退表现

```
清晰 (100-80%)    轻微模糊 (80-50%)    重度模糊 (50-20%)    遗忘 (20-0%)
┌───┐             ┌───┐                ┌───┐                ???
│   │             │   │                │···│
│ A │             │ ░ │                │░░░│
│   │             │   │                │···│
└───┘             └───┘                └───┘
线条清晰          线条变淡/抖动         只剩轮廓             完全空白
```

### 阻止衰退的方法

1. **定期回访** - 最简单的办法
2. **墨水绘制** - 羽毛笔绘制的地图不衰减
3. **记忆强化技能** - 减缓衰退速度
4. **锚定地标** - 重要地标记忆不会衰退
5. **魔法卷轴** - 永久记录

## 5. 纸张与背包系统

### 纸张类型

| 类型 | 格数 | 耐久 | 特性 |
|------|------|------|------|
| 粗糙纸张 | 20×20 | 易破损 | 遇水即毁 |
| 羊皮纸 | 30×30 | 耐磨损 | 标准纸张 |
| 防水纸 | 25×25 | 防水 | 潮湿区域必备 |
| 魔法纸 | 40×40 | 永不破损 | 可叠加绘制 |

### 背包限制

```yaml
基础背包: 10格
地图纸张: 每份占1格（可叠加5张同类型）
绘制工具: 每种占1-2格
光源: 火把占1格（可叠加3个），提灯占2格
补给: 食物、水等
```

### 纸张管理策略

- **分层记录** - 每层迷宫用不同纸张
- **区域划分** - 将大地图分割记录
- **优先级** - 只记录重要区域
- **备份策略** - 关键路径多抄一份

## 8. 进阶机制

当玩家获得多张地图碎片时，需要手动拼接：
- 旋转对齐

- 发现隐藏连接

## 9. v1.1 新增：心理状态系统

### 9.1 压力值机制

```typescript
interface MentalState {
  stressLevel: number;      // 0-100
  hallucinationChance: number; // 基于压力的概率
  drawingAccuracyPenalty: number; // 压力对绘图的影响
}
```

**压力来源:**

| 情况 | 压力增加 | 说明 |
|------|----------|------|
| 在黑暗中每5分钟 | +5 | 未知带来恐惧 |
| 迷路时每分钟 | +10 | 恐慌累积 |
| 发现尸体/遗迹 | +15 | 心理冲击 |
| 被追逐时 | +20 | 生存压力 |
| 地图出现矛盾 | +8 | 认知失调 |
| 体力低于20% | +10 | 生理影响心理 |

**压力影响:**

| 压力范围 | 状态 | 效果 |
|----------|------|------|
| 0-30 | 冷静 | 无惩罚，可能+5%精度 |
| 30-50 | 紧张 | 绘图误差+10% |
| 50-70 | 焦虑 | 误差+20%，手抖动画 |
| 70-85 | 恐慌 | 误差+35%，开始出现幻觉 |
| 85-100 | 崩溃 | 误差+50%，频繁幻觉，视野闪烁 |

### 9.2 幻觉系统

**幻觉类型:**

1. **虚假通道** - 在墙上显示不存在的门
2. **错误标记** - 已探索区域显示为未探索
3. **幽灵敌人** - 看到不存在的怪物
4. **地图扭曲** - 已绘制的线条轻微偏移
5. **声音幻听** - 听到脚步声、低语

**幻觉检测:**
- 使用测绘仪可以验证
- 与其他玩家地图对比
- 高压力消退后自动发现

### 9.3 压力恢复

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 安全区域休息 | -15/分钟 | 无威胁 |
| 完成准确绘图 | -20 | 准确度>90% |
| 镇定药水 | -30 | 消耗品 |
| 烟草/草药 | -10 | 轻微副作用 |
| 发现出口 | -40 | 希望恢复 |

## 10. v1.1 新增：环境叙事系统

### 10.1 遗迹痕迹

**古老符号:**
- 墙壁上随机出现的古代文字
- 考古学家职业可以解读
- 解读成功获得提示（出口方向、危险警告）

**前人标记:**
```
标记类型:
- 箭头标记 → 方向指示（可能正确也可能误导）
- 骷髅标记 ☠ 危险警告
- 问号标记 ❓ 未探索
- 叉号标记 ✗ 死胡同
- 特殊符号 需要解读
```

**可信度机制:**
- 不同前人的标记可信度不同
- 某些是故意误导
- 通过考古学技能判断真伪

### 10.2 日记残页

**收集系统:**
- 散落在迷宫各处
- 逐步揭示迷宫历史和背景故事
- 收集完整可解锁隐藏区域

**内容示例:**
```
[第1页] "我们以为这只是普通的遗迹..."
[第5页] "迷宫在变化，我发誓那条路昨天不在那里"
[第12页] "我看到了他们，那些先我们一步的人，但他们已经不..."
```

### 10.3 动态环境

**周期性变化:**
- **潮汐区域** - 某些通道定时被水淹没/露出
- **机关重置** - 每游戏日重置一次
- **月相影响** - 特定月相时隐藏区域开放

**天气系统:**
- 露天区域受天气影响
- 暴雨改变地形
- 大雾进一步限制视野

## 11. v1.1 新增：地图验证机制

### 11.1 矛盾检测

**检测类型:**

```typescript
enum MapContradiction {
  WALL_PATH_MISMATCH,    // 画成通道实际有墙
  CONNECTIVITY_ERROR,    // 连通性错误
  DISTANCE_MISMATCH,     // 距离不符
  IMPOSSIBLE_GEOMETRY,   // 几何不可能
}
```

**触发条件:**
- 玩家尝试穿过自己画的"通道"被墙挡住
- 两个已探索区域无法按地图连通
- 测量距离与地图标记不符

### 11.2 修正成本

| 修正方式 | 成本 | 限制 |
|----------|------|------|
| 铅笔修改 | 橡皮耐久-1 | 仅铅笔绘制可改 |
| 墨水覆盖 | 无法覆盖 | 必须重绘 |
| 贴修正纸 | 纸张-1 | 仅限粗糙纸 |
| 重绘区域 | 时间+体力 | 最彻底但成本高 |

### 11.3 "地图混乱"状态

当矛盾积累过多时触发:
- 地图界面出现视觉扭曲
- 无法信任自己的地图
- 必须找到参考点重新校准
- 可能触发特殊剧情

## 15. v1.2 新增：死亡与传承系统

### 15.1 死亡惩罚

- 丢失当前携带物品
- 地图知识保留但精度-30%

- 回收前角色地图（最宝贵）

### 15.3 幽灵模式

死亡后短暂幽灵形态:
- 可观察周围但不能互动
- 可留下标记警告后来者
- 帮助下一个自己
