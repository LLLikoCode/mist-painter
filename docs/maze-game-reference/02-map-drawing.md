# 《迷雾绘者》- 地图绘制系统

> **系统:** Map Drawing System  
> **重要性:** ⭐ 核心机制  
> **设计理念:** 玩家不是"看到"地图，而是"绘制"地图

---

## 1. 视野与迷雾系统

### 1.1 三层视野模型

```
[真实视野]  ← 玩家实际能看到的范围（3-5格）
    ↓
[记忆地图]  ← 玩家绘制在纸上的地图（可能有误差）
    ↓
[迷雾区域]  ← 未探索或已遗忘的区域
```

### 1.2 真实视野 (True Sight)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 基础视野半径 | 3格 | 不受任何影响时的视野 |
| 火把照明 | +2格 | 手持火把时的加成 |
| 提灯照明 | +3格 | 更稳定的光源 |
| 黑暗环境 | -1格 | 无光源时的惩罚 |
| 迷雾区域 | -1格 | 特殊区域的惩罚 |

### 1.3 视野边缘效果

视野边缘不是"硬切"，而是渐进模糊：

```
  清晰    轻微模糊    重度模糊    不可见
   ●  →    ◐    →    ◑    →    ○
  100%     70%        30%        0%
```

## 2. 绘制工具系统

### 2.1 工具类型详解

#### 铅笔 (Pencil)
```yaml
名称: 探险者铅笔
耐久: 100笔划
特性:
  - 可擦除修改
  - 绘制速度快
  - 精度中等 (误差率: 15%)
缺点:
  - 容易磨损
  - 遇水模糊（潮湿区域风险）
获取: 初始装备，商店常见
```

#### 羽毛笔+墨水 (Quill & Ink)
```yaml
名称: 学者羽毛笔
耐久: 50笔划（墨水限制）
特性:
  - 永久记录，无法修改
  - 精度高 (误差率: 5%)
  - 可绘制细节标记
缺点:
  - 墨水有限
  - 错误无法修正
获取: 中层迷宫宝箱，商店稀有
```

#### 测绘仪 (Surveying Tool)
```yaml
名称: 精密测绘仪
耐久: 无限（但使用耗时）
特性:
  - 极高精度 (误差率: 1%)
  - 可测量距离和角度
  - 发现隐藏结构
缺点:
  - 占用2格背包
  - 使用时不能移动
  - 绘制速度极慢
获取: 深层迷宫，任务奖励
```

#### 魔法卷轴 (Magic Scroll)
```yaml
名称: 记忆卷轴
耐久: 一次性
特性:
  - 自动记录当前区域
  - 100%准确
  - 永久不衰退
缺点:
  - 极其稀有
  - 范围有限（当前房间/区域）
获取: 隐藏宝箱，特殊事件
```

### 2.2 工具对比表

| 工具 | 精度 | 速度 | 可修改 | 耐久 | 背包占用 |
|------|------|------|--------|------|----------|
| 铅笔 | ★★★ | ★★★★★ | ✓ | 100划 | 1格 |
| 羽毛笔 | ★★★★ | ★★★ | ✗ | 50划 | 1格 |
| 测绘仪 | ★★★★★ | ★ | - | 无限 | 2格 |
| 魔法卷轴 | ★★★★★ | ★★★★★ | - | 1次 | 1格 |

## 3. 地图精度与误差机制

### 3.1 误差产生原因

```
误差率 = 基础误差 + 环境惩罚 + 移动惩罚 + 工具修正
```

| 因素 | 误差加成 | 说明 |
|------|----------|------|
| 基础误差 | +10% | 任何绘制都有基础误差 |
| 移动中绘制 | +15% | 边走边画 |
| 黑暗环境 | +20% | 光源不足 |
| 迷雾区域 | +25% | 感知扭曲 |
| 魔法干扰 | +30% | 特殊区域 |
| 疲劳状态 | +10% | 体力低于30% |

### 3.2 误差表现形式

#### 距离误差
```
实际:    绘制结果:
┌───┐    ┌───┐
│   │    │   │
│ A ═══ B    │ A ═ B  (距离缩短)
│   │    │   │
└───┘    └───┘
```

#### 角度误差
```
实际:    绘制结果:
    B         B
   /          |
  /           |
 A            A─── (角度变直)
```

#### 连通性误差
```
实际:    绘制结果:
A ═══ B    A ═══ B
║     ║    ║     ║
C ═══ D    C     D (遗漏通道)
```

#### 幻觉误差（深层迷宫特有）
```
实际:    绘制结果:
┌───┐    ┌───┬───┐
│   │    │   │幻觉│
│ A │    │ A │通道│
│   │    │   │    │
└───┘    └───┴───┘
        (绘制了不存在的通道)
```

### 3.3 误差检测与修正

玩家可以通过以下方式发现误差：
- **回访验证** - 回到已绘制区域，发现不一致
- **测绘仪校准** - 使用测绘仪重新测量
- **古代地图碎片** - 对照真实地图
- **地标确认** - 特殊地标不会出错

## 4. 记忆衰退系统

### 4.1 衰退机制

```
记忆清晰度 = 100% - (经过时间 / 衰退周期) × 衰退率
```

| 层级 | 衰退周期 | 衰退率 | 说明 |
|------|----------|--------|------|
| 表层 | 30分钟 | 20% | 记忆相对稳定 |
| 中层 | 20分钟 | 35% | 开始明显模糊 |
| 深层 | 10分钟 | 50% | 快速遗忘 |
| 混沌层 | 5分钟 | 80% | 几乎无法记住 |

### 4.2 衰退表现

```
清晰 (100-80%)    轻微模糊 (80-50%)    重度模糊 (50-20%)    遗忘 (20-0%)
┌───┐             ┌───┐                ┌───┐                ???
│   │             │   │                │···│
│ A │             │ ░ │                │░░░│
│   │             │   │                │···│
└───┘             └───┘                └───┘
线条清晰          线条变淡/抖动         只剩轮廓             完全空白
```

### 4.3 阻止衰退的方法

1. **定期回访** - 最简单的办法
2. **墨水绘制** - 羽毛笔绘制的地图不衰减
3. **记忆强化技能** - 减缓衰退速度
4. **锚定地标** - 重要地标记忆不会衰退
5. **魔法卷轴** - 永久记录

## 5. 纸张与背包系统

### 5.1 纸张类型

| 类型 | 格数 | 耐久 | 特性 |
|------|------|------|------|
| 粗糙纸张 | 20×20 | 易破损 | 遇水即毁 |
| 羊皮纸 | 30×30 | 耐磨损 | 标准纸张 |
| 防水纸 | 25×25 | 防水 | 潮湿区域必备 |
| 魔法纸 | 40×40 | 永不破损 | 可叠加绘制 |

### 5.2 背包限制

```yaml
基础背包: 10格
地图纸张: 每份占1格（可叠加5张同类型）
绘制工具: 每种占1-2格
光源: 火把占1格（可叠加3个），提灯占2格
补给: 食物、水等
```

### 5.3 纸张管理策略

- **分层记录** - 每层迷宫用不同纸张
- **区域划分** - 将大地图分割记录
- **优先级** - 只记录重要区域
- **备份策略** - 关键路径多抄一份

### 7.1 地图界面布局

```
┌─────────────────────────────────────────┐
│  [工具栏] 铅笔 | 羽毛笔 | 测绘仪 | 橡皮   │
├─────────────────────────────────────────┤
│                                         │
│         [地图绘制区域]                   │
│         (羊皮纸质感背景)                  │
│                                         │
│         玩家位置: ●                      │
│         视野范围: ○                      │
│                                         │
├─────────────────────────────────────────┤
│  [标记栏] ⚠️ 💰 🚪 💀 ❓ ✓ 🔄 ⬆️ ⬇️     │
```

### 7.2 绘制交互

1. **自动模式** - 边走边自动记录（精度低）
2. **手动模式** - 停下仔细绘制（精度高）
3. **测绘模式** - 使用测绘仪精确测量

### 7.3 视觉反馈

- **绘制时** - 墨水/铅笔笔触动画
- **出错时** - 轻微震动 + 线条抖动
- **衰退时** - 线条逐渐变淡
- **发现错误** - 红色闪烁提示

## 8. 进阶机制

### 8.1 地图拼接

当玩家获得多张地图碎片时，需要手动拼接：
- 旋转对齐
- 边缘匹配
- 发现隐藏连接

### 8.2 地图交易

多人模式下：
- 可以交换地图副本
- 副本可能有误差
- 需要验证对方地图的准确性

### 8.3 古代地图解读

找到的古代地图：
- 使用已失传的文字
- 需要翻译/解读
- 可能故意包含误导信息
