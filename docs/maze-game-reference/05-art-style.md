# 《迷雾绘者》- 美术风格指南

> **系统:** Art & Visual Design  
> **设计理念:** 视觉是玩家第一眼看到的。羊皮纸、墨水、迷雾——古老而神秘。

---

## 1. 整体风格

### 1.1 关键词

```
古老 · 神秘 · 手绘 · 沉浸 · 温暖
```

### 1.2 参考风格

- **地图界面**: 中世纪航海图、羊皮纸地图
- **游戏世界**: 像素艺术 + 手绘质感
- **UI**: 复古纸张、墨水书写
- **氛围**: 地牢探险游戏 (Darkest Dungeon, Loop Hero)

## 3. 地图绘制界面

### 3.1 羊皮纸质感

```
视觉要素:
┌─────────────────────────────────────┐
│  ▓▓ 不规则边缘（撕裂效果）            │
│ ▓▓▓                                 │
│▓▓▓▓  泛黄底色 + 污渍                 │
│ ▓▓▓    ～～ 轻微褶皱纹理              │
│  ▓▓        · 老化斑点                │
│                                     │
│    [手绘地图内容]                    │
│                                     │
│  ═══════════════════               │
│  边缘装饰（可选复古花纹）              │
└─────────────────────────────────────┘
```

### 3.2 绘制笔触效果

#### 铅笔效果
- 颜色: #666666
- 线条: 略带抖动，不完全直
- 透明度: 80%
- 特点: 可擦除，有橡皮擦痕迹

#### 墨水效果
- 颜色: #1a1a1a
- 线条: 流畅但偶有飞白
- 透明度: 100%
- 特点: 永久，边缘可能有轻微晕染

#### 测绘仪效果
- 颜色: #000000
- 线条: 精确直线
- 透明度: 100%
- 特点: 有测量标记（小十字、数字）

### 3.3 记忆衰退效果

```
清晰度 100%:  ████████████████████  深黑，清晰
清晰度 70%:   █████████████████░░░  略淡，轻微模糊
清晰度 50%:   █████████████░░░░░░░  明显变淡
清晰度 30%:   ███████░░░░░░░░░░░░░  只剩轮廓
清晰度 10%:   ██░░░░░░░░░░░░░░░░░░  几乎消失
```

实现方式：
- 透明度递减
- 添加轻微模糊滤镜
- 线条变细
- 颜色向纸张色靠近

## 4. 游戏世界视觉

### 4.1 单元格设计

#### 墙壁
```
██████  深色方块
██░░██  添加细微纹理（石砖感）
██████
```

#### 通道
```
░░░░░░  浅色，羊皮纸色
░    ░  可能有细微纹理
░░░░░░
```

#### 特殊地形

| 类型 | 视觉表现 |
|------|----------|
| 入口 | 绿色光晕，拱门形状 |
| 出口 | 红色光晕，楼梯向下 |
| 隐藏门 | 与墙壁相同，但仔细看有缝隙 |
| 传送点 | 蓝色漩涡，轻微动画 |
| 陷阱 | 地板上有细微不同（裂纹/颜色） |

### 4.2 视野效果

#### 迷雾效果
```
未探索区域:  纯黑 #0a0a0a
已探索但不可见:  深色遮罩 rgba(0,0,0,0.7)
视野边缘:  渐变过渡
```

#### 光源效果
```
火把照明:
  - 中心: 明亮温暖
  - 边缘: 逐渐变暗
  - 轻微闪烁动画
  
提灯照明:
  - 更稳定的光源
  - 范围更大
  - 边缘更柔和
```

### 4.3 粒子效果

| 效果 | 描述 |
|------|------|
| 火把火星 | 少量橙色粒子向上飘 |
| 迷雾粒子 | 缓慢飘动的半透明粒子 |
| 绘制动画 | 墨水/铅笔的绘制轨迹 |
| 记忆衰退 | 粒子消散效果 |
| 传送激活 | 蓝色漩涡粒子 |

## 6. 动画规范

### 6.1 过渡时间

| 动画类型 | 时长 | 缓动 |
|----------|------|------|
| 界面切换 | 300ms | ease-out |
| 按钮悬停 | 150ms | ease |
| 地图绘制 | 实时 | linear |
| 记忆衰退 | 2000ms | ease-in |
| 光源闪烁 | 100ms | random |
| 粒子效果 | 500-2000ms | ease-out |

### 6.2 关键动画

#### 地图绘制
```
1. 玩家移动到新格子
2. 如果是新区域:
   - 迷雾消散 (300ms)
   - 如果是自动绘制，显示绘制动画
3. 绘制动画:
   - 线条从起点延伸到终点
   - 根据工具类型有不同的速度
```

#### 记忆衰退
```
1. 触发条件: 时间经过
2. 视觉效果:
   - 线条逐渐变淡 (2s)
   - 添加轻微模糊
   - 可选: 纸张上的水渍效果
```

#### 光源效果
```
火把:
  - 轻微位置抖动 (模拟手持)
  - 亮度随机波动 (0.9-1.1)
  - 偶尔火星粒子

提灯:
  - 稳定的光源
  - 轻微摇摆 (如果玩家移动)
```
