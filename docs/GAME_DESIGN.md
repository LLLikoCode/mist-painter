## 核心玩法机制

### 游戏概念
玩家扮演一位拥有特殊能力的"迷雾绘者"，可以在被迷雾笼罩的世界中绘制路径、揭示隐藏的秘密，并解开各种谜题。

### 核心机制
1. **迷雾绘制**: 玩家可以使用特殊能力在迷雾中绘制可见区域
2. **路径探索**: 通过绘制迷雾来创造可行走的路径
3. **谜题解决**: 利用迷雾机制解开各种环境谜题
4. **资源管理**: 迷雾绘制能力有限，需要策略性使用

## 技术架构

### 游戏引擎
- **引擎**: Godot 4.x
- **脚本语言**: GDScript
- **目标平台**: PC, Web

### 系统架构
```
├── Core (核心系统)
│   ├── GameStateManager (游戏状态管理)
│   ├── EventBus (事件总线)
│   └── SaveManager (存档系统)
├── Gameplay (游戏玩法)
│   ├── MistPaintingSystem (迷雾绘制系统)
│   ├── PuzzleController (谜题控制器)
│   └── GameController (游戏主控制器)
├── Player (玩家)
│   └── PlayerController (玩家控制器)
├── UI (用户界面)
│   ├── UIManager (UI管理器)
│   ├── MainMenu (主菜单)
│   └── NavigationManager (导航管理)
├── Audio (音频)
│   └── AudioManager (音频管理器)
└── Achievements (成就)
    └── AchievementManager (成就管理器)
```
