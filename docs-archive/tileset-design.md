## 1. 设计概述

### 1.1 设计目标

本文档定义《迷雾绘者》游戏场景的 tileset 资源规格，确保场景美术资源与角色 sprite 风格统一，支持多层渲染系统，并满足游戏解谜机制的需求。

### 1.2 设计原则

- **风格统一**: 与角色 sprite (32x48px) 保持一致的像素艺术风格
- **模块化设计**: 支持灵活组合，减少重复资源
- **多层兼容**: 支持背景层、地面层、前景层的分离渲染
- **解谜友好**: 可交互 tiles 具有清晰的视觉区分
- **性能优化**: 合理使用 tile 数量，避免过度复杂

### 1.3 参考文档

- [美术风格指南](../art-style-guide.md) - 整体视觉风格定义
- [角色 Sprite 规格](../character-sprite-spec.md) - 角色尺寸与配色参考

## 2. Tileset 规格定义

### 2.1 基础规格

| 参数 | 值 | 说明 |
|------|-----|------|
| **Tile 尺寸** | 32x32 像素 | 与角色 sprite 比例协调 (1 tile = 角色下半身) |
| **色彩深度** | 32-bit RGBA | 支持透明通道 |
| **文件格式** | PNG | 无损压缩，透明背景 |
| **像素风格** | 像素艺术 | 与角色风格统一 |
| **Tileset 尺寸** | 512x512 像素 (标准) | 16x16 tiles，可扩展至 1024x1024 |

### 2.2 为什么选择 32x32

```
角色与 Tile 比例示意:

    +--------+
    |  头部  |  <- 约 16px (上半部分)
    |        |
    +--------+
    |  身体  |  <- 约 16px
    |        |
    +--------+
    |  腿部  |  <- 约 16px (下半部分)
    +--------+
    
    +--------+
    |        |  <- 1 Tile (32x32px)
    |  TILE  |
    |        |
    +--------+
    
角色高度 = 1.5 Tiles (48px)
```

**优势**:
- 角色可自然站立在 1 个 tile 上
- 通道宽度 1 tile 时角色刚好通过
- 与主流像素游戏标准一致

## 3. 环境区域设计

### 3.1 区域概览

游戏包含 4 个主要环境区域，每个区域有独特的视觉风格：

| 区域 | 主题 | 色调 | 氛围 |
|------|------|------|------|
| **表层遗迹** | 遗迹入口 | 暖黄 #d4c4a8 + 灰褐 | 相对明亮，风化痕迹 |
| **中层回廊** | 古代回廊 | 灰绿 #5a6a5a + 褐 | 潮湿，苔藓，古老 |
| **深层深渊** | 迷失深渊 | 深蓝 #2a3a5a + 黑 | 黑暗，神秘符文 |
| **混沌空间** | 异空间 | 紫 #4a3a5a + 黑 | 扭曲，不稳定感 |

### 3.2 表层遗迹 (Surface Ruins)

#### 视觉特征
- **主色调**: #d4c4a8 (羊皮纸深) + #8a7a6a (风化灰)
- **特点**: 阳光照射，风化痕迹明显，相对明亮
- **材质**: 砂岩、风化石砖

#### Tile 类型
| 类型 | 描述 | 示例 |
|------|------|------|
| 地面 | 风化石板，有裂纹 | floor_surface_*.png |
| 墙壁 | 砂岩砖墙，风化边缘 | wall_surface_*.png |
| 装饰 | 碎石、枯草、阳光光斑 | deco_surface_*.png |

### 3.3 中层回廊 (Middle Corridors)

#### 视觉特征
- **主色调**: #5a6a5a (灰绿) + #6a5a4a (潮湿褐)
- **特点**: 潮湿阴暗，苔藓覆盖，古老感
- **材质**: 青石砖、苔藓石

#### Tile 类型
| 类型 | 描述 | 示例 |
|------|------|------|
| 地面 | 青苔石板，湿润反光 | floor_middle_*.png |
| 墙壁 | 苔藓覆盖的石墙 | wall_middle_*.png |
| 装饰 | 苔藓、水滴、菌类 | deco_middle_*.png |

### 3.4 深层深渊 (Deep Abyss)

#### 视觉特征
- **主色调**: #2a3a5a (深蓝) + #1a1a2a (暗影)
- **特点**: 黑暗神秘，神秘符文，压抑感
- **材质**: 黑曜石、符文石板

#### Tile 类型
| 类型 | 描述 | 示例 |
|------|------|------|
| 地面 | 黑曜石地面，微光符文 | floor_deep_*.png |
| 墙壁 | 符文石墙，神秘图案 | wall_deep_*.png |
| 装饰 | 符文、水晶、暗影 | deco_deep_*.png |

### 3.5 混沌空间 (Chaos Realm)

#### 视觉特征
- **主色调**: #4a3a5a (紫) + #2a1a3a (暗紫)
- **特点**: 空间扭曲，不稳定，超现实
- **材质**: 虚空结晶、扭曲空间

#### Tile 类型
| 类型 | 描述 | 示例 |
|------|------|------|
| 地面 | 虚空地面，流动效果 | floor_chaos_*.png |
| 墙壁 | 扭曲空间边界 | wall_chaos_*.png |
| 装饰 | 虚空粒子、扭曲光效 | deco_chaos_*.png |
