# 迷雾绘者 (Mist Painter) 特效动画设计文档

## 概述
本文档描述游戏中使用的所有特效动画资源，包括设计意图、技术参数和使用场景。

---

## 1. 迷雾绘制特效 (mist_draw)

### 用途
当玩家使用画笔在迷雾区域绘制时触发的粒子效果，表现迷雾被"激活"并逐渐消散的过程。

### 视觉设计
- **风格**: 手绘水墨风格，边缘带有不规则的笔触感
- **颜色**: 淡紫色到透明的渐变 (#9B7EDE → transparent)
- **形态**: 从中心向外扩散的圆形波纹，带有细小的粒子飞溅

### 动画参数
- **帧数**: 12帧
- **帧率**: 12 FPS
- **时长**: 1.0秒
- **循环方式**: 单次播放 (oneshot)
- **尺寸**: 128x128 像素

### 关键帧描述
| 帧 | 描述 |
|---|------|
| 1-3 | 中心点出现，淡紫色光晕 |
| 4-7 | 波纹向外扩散，粒子飞溅 |
| 8-10 | 波纹继续扩散，颜色变淡 |
| 11-12 | 逐渐消失，留下微弱残影 |

## 2. 迷雾擦除特效 (mist_erase)

### 用途
当玩家使用橡皮擦工具擦除迷雾时触发，表现迷雾被"擦除"的消散效果。

### 视觉设计
- **风格**: 手绘风格，边缘呈撕裂状
- **颜色**: 深灰色到透明的渐变 (#4A4A4A → transparent)
- **形态**: 从擦除位置向四周飞散的小碎片

### 动画参数
- **帧数**: 10帧
- **帧率**: 15 FPS
- **时长**: 0.67秒
- **循环方式**: 单次播放 (oneshot)
- **尺寸**: 64x64 像素

### 关键帧描述
| 帧 | 描述 |
|---|------|
| 1-2 | 擦除点出现小碎片 |
| 3-6 | 碎片向四周飞散 |
| 7-10 | 碎片逐渐变小并消失 |

## 3. 角色脚步特效 (footstep)

### 用途
角色移动时在脚下产生的轻微尘埃效果，增强移动感。

### 视觉设计
- **风格**: 简约手绘尘埃
- **颜色**: 浅灰色半透明 (#CCCCCC, alpha 0.4)
- **形态**: 两个小椭圆，左右交替出现

### 动画参数
- **帧数**: 8帧
- **帧率**: 12 FPS
- **时长**: 0.67秒
- **循环方式**: 单次播放 (oneshot)
- **尺寸**: 64x64 像素

### 关键帧描述
| 帧 | 描述 |
|---|------|
| 1-2 | 左脚踏点出现小尘埃 |
| 3-4 | 左尘埃扩散 |
| 5-6 | 右脚踏点出现小尘埃 |
| 7-8 | 右尘埃扩散并消失 |

- **循环方式**: 单次播放 (oneshot)

| 5-6 | 内圈再次显示 |
| 7-8 | 外圈渐隐，回到初始状态 |

## 技术规范

### 文件命名规范
```
{effect_name}_{frame_number}.png
例如: mist_draw_01.png, mist_draw_02.png, ...
```

### 图像格式
- **格式**: PNG
- **通道**: RGBA (带透明通道)
- **位深**: 8-bit per channel
- **压缩**: 无损压缩

### 目录结构
```
assets/effects/
├── effects_design_doc.md      # 本文档
├── effects_config.json        # 配置文件
├── mist_draw/                 # 迷雾绘制特效
│   ├── mist_draw_01.png
│   ├── mist_draw_02.png
│   └── ...
├── mist_erase/                # 迷雾擦除特效
│   ├── mist_erase_01.png
│   └── ...
├── footstep/                  # 脚步特效
├── door_open/                 # 开门特效
├── mechanism_trigger/         # 机关触发特效
├── ui_transition/             # UI转场特效
├── collectible/               # 收集物特效
└── hint_ring/                 # 提示光圈特效
```
