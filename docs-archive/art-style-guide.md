## 1. 整体视觉风格定义

### 1.1 核心关键词

```
古老 · 神秘 · 手绘 · 沉浸 · 温暖 · 羊皮纸质感
```

### 1.2 艺术风格方向

**主风格**: 手绘水彩 + 像素艺术融合

- **地图界面**: 中世纪航海图、羊皮纸地图风格
- **游戏世界**: 像素艺术为基础，融合手绘质感
- **UI**: 复古纸张、墨水书写风格
- **氛围参考**: 《Darkest Dungeon》《Loop Hero》《Gris》

### 1.3 光影处理方式

#### 光源类型
| 光源 | 色温 | 特点 |
|------|------|------|
| 火把 | 暖橙 #ffaa44 | 闪烁、不稳定、范围小 |
| 提灯 | 暖黄 #ffcc66 | 稳定、范围大、边缘柔和 |
| 环境光 | 冷灰 #4a5560 | 未探索区域的微光 |
| 魔法光 | 淡蓝 #88ccff | 特殊区域、传送点 |

#### 光影原则
- **动态光源**: 玩家手持光源产生实时阴影
- **迷雾遮罩**: 未探索区域使用渐变遮罩，非硬切
- **边缘柔化**: 视野边缘使用渐变过渡，增强沉浸感
- **记忆衰退**: 已探索但久未回访的区域逐渐变淡

### 1.4 画面构图原则

#### 地图绘制界面构图
```
┌─────────────────────────────────────────┐
│  [顶部工具栏 - 横向排列]                 │
├─────────────────────────────────────────┤
│                                         │
│         [中央地图区域]                   │
│         羊皮纸背景 + 手绘线条            │
│         玩家位置居中或偏下               │
│                                         │
├─────────────────────────────────────────┤
│  [底部状态栏 - 资源/耐久/图层信息]       │
└─────────────────────────────────────────┘
```

#### 游戏世界构图
- **视角**: 俯视2D，略带倾斜（可选）
- **视野**: 以玩家为中心，圆形视野范围
- **层次**: 前景（玩家）→ 中景（迷宫）→ 背景（迷雾）

## 3. 角色设计规范

### 3.1 主角视觉特征

#### 基础设定
- **体型**: 中等比例，便于像素表现
- **风格**: 像素艺术 + 轮廓光
- **配色**: 深色调为主，便于在浅色地图背景中识别

#### 视觉元素
```
探险者形象:
┌─────────────────┐
│    ◠◠         │  ← 兜帽遮住大部分面部，仅露眼睛
│   /███\       │  ← 深色斗篷
│   │███│       │  ← 背包/绘制工具包
│   /   \       │  ← 长靴
│  /     \      │
└─────────────────┘
```

#### 配色方案
| 部位 | 颜色 | 说明 |
|------|------|------|
| 斗篷 | `#2d2d2d` | 深灰/黑色 |
| 兜帽内 | `#1a1a1a` | 墨黑，神秘感 |
| 眼睛 | `#c9a227` | 金色微光 |
| 背包 | `#5a4a3a` | 皮革色 |
| 靴子 | `#3d2817` | 深褐色 |

### 3.2 NPC设计原则

#### 设计规范
- **剪影识别**: 每个NPC有独特剪影
- **色彩区分**: 使用强调色区分不同类型
- **比例统一**: 与主角保持相同比例

#### NPC类型配色
| 类型 | 主色 | 特点 |
|------|------|------|
| 流浪商人 | `#c9a227` + `#5a4a3a` | 金色装饰，商人气质 |
| 前探险者 | `#4a6fa5` + `#2d2d2d` | 蓝色披风，学者感 |
| 神秘学者 | `#8b3a3a` + `#1a1a1a` | 暗红+黑，神秘感 |
| 遗迹守护者 | `#4a7c59` + `#c9b896` | 绿色+旧纸色，古老感 |

### 3.3 角色比例与风格统一性

#### 比例规范
```
标准角色尺寸: 32×48 像素
┌──────────────┐
│   头部: 12px │
│   身体: 16px │
│   腿部: 20px │
└──────────────┘
```

#### 风格统一原则
- 所有角色使用相同像素密度
- 轮廓线粗细统一（1-2像素）
- 阴影方向统一（左上方光源）
- 避免过度细节，保持可读性

### 3.4 表情/动作表现方式

#### 表情系统
由于像素限制，通过以下方式表现：
- **眼睛形状变化**: 圆眼(正常) → 横线(闭眼) → 竖线(惊讶)
- **身体姿态**: 直立(正常) → 前倾(紧张) → 后仰(受伤)
- **道具动画**: 手持物品抖动表现紧张

#### 动作设计
| 动作 | 帧数 | 描述 |
|------|------|------|
| 待机 | 2帧 | 轻微呼吸起伏 |
| 行走 | 4帧 | 左右摇摆，斗篷飘动 |
| 绘制 | 6帧 | 手臂挥动，笔触效果 |
| 受伤 | 3帧 | 后仰，红色闪烁 |
| 死亡 | 5帧 | 倒地，淡出 |

## 4. 场景与环境设计

### 4.1 关卡场景风格

#### 整体风格
- **古老遗迹**: 石质结构，风化痕迹
- **神秘氛围**: 迷雾笼罩，光线昏暗
- **手绘质感**: 墙壁纹理带有手绘笔触

#### 层级风格差异
| 层级 | 主题 | 色调 | 特点 |
|------|------|------|------|
| 表层 | 遗迹入口 | 暖黄+灰 | 相对明亮，风化痕迹 |
| 中层 | 古代回廊 | 灰绿+褐 | 潮湿，苔藓，古老 |
| 深层 | 迷失深渊 | 深蓝+黑 | 黑暗，神秘符文 |
| 混沌层 | 异空间 | 紫+黑 | 扭曲，不稳定感 |

### 4.2 背景层次处理

#### 三层背景系统
```
前景层: 玩家、敌人、可交互物体
中景层: 迷宫墙壁、地板、通道
背景层: 迷雾、环境氛围、远景
```

#### 景深效果
- **前景**: 100% 不透明度，清晰
- **中景**: 100% 不透明度，标准清晰度
- **背景**: 60-80% 不透明度，轻微模糊

### 4.3 迷雾/梦境效果表现

#### 迷雾效果
```
技术实现:
┌─────────────────────────────────┐
│  未探索: 纯黑遮罩 #0a0a0a       │
│  已探索不可见: 70%黑色遮罩      │
│  视野边缘: 渐变过渡             │
│  视野内: 无遮罩                 │
└─────────────────────────────────┘
```

#### 梦境效果（特殊区域）
- **色彩偏移**: 整体色调向紫/蓝偏移
- **边缘扭曲**: 轻微波浪变形
- **粒子效果**: 缓慢上升的微光粒子
- **记忆衰退**: 已绘制线条轻微抖动

### 4.4 环境交互元素设计

#### 可交互元素视觉反馈
| 元素 | 常态 | 悬停 | 激活 |
|------|------|------|------|
| 门 | 静态 | 高亮边框 | 打开动画 |
| 宝箱 | 关闭 | 发光边缘 | 开启+粒子 |
| 机关 | 静态 | 微光 | 触发效果 |
| 传送点 | 缓慢旋转 | 加速旋转 | 传送动画 |

#### 特殊地形表现
| 类型 | 视觉表现 |
|------|----------|
| 入口 | 绿色光晕 `#4a7c59`，拱门形状 |
| 出口 | 红色光晕 `#8b3a3a`，楼梯向下 |
| 隐藏门 | 与墙壁相同，仔细看有细微缝隙 |
| 传送点 | 蓝色漩涡 `#4a6fa5`，轻微动画 |
| 陷阱 | 地板上有细微不同（裂纹/颜色） |

## 6. 特效与动画风格

### 6.1 粒子效果风格

#### 总体风格
- **手绘感**: 粒子带有手绘笔触
- **柔和边缘**: 避免硬边，使用渐变
- **有限数量**: 保持性能，注重质量

#### 粒子类型
| 效果 | 颜色 | 运动 | 生命周期 |
|------|------|------|----------|
| 火把火星 | `#ffaa44` | 向上飘，随机摆动 | 0.5-1.5s |
| 迷雾粒子 | `#2a2520` | 缓慢水平飘动 | 3-5s |
| 绘制轨迹 | `#1a1a1a` | 跟随笔触 | 实时 |
| 记忆衰退 | `#d4c4a8` | 消散 | 2s |
| 传送漩涡 | `#4a6fa5` | 旋转收缩 | 1-2s |

### 6.2 转场动画设计

#### 场景切换
```
类型: 墨水晕染
时长: 500ms
效果: 黑色墨水从中心扩散，覆盖全屏
音效: 纸张翻动 + 墨水声
```

#### 层级切换
```
类型: 下落/上升
时长: 800ms
效果: 玩家向下/向上移动，画面跟随
特效: 粒子拖尾，速度线
```

#### UI界面切换
```
类型: 纸张滑动
时长: 300ms
缓动: ease-out
效果: 新界面从下方滑入，旧界面淡出
```

### 6.3 解谜成功/失败反馈

#### 成功反馈
```
视觉:
  - 金色粒子爆发
  - 地图区域高亮
  - 轻微震动效果

音效:
  - 神秘提示音
  - 纸张展开声

动画:
  - 持续时间: 1.5s
  - 缓动: ease-out
```

#### 失败反馈
```
视觉:
  - 红色闪烁
  - 屏幕震动
  - 墨水飞溅效果

音效:
  - 低沉警告音
  - 纸张撕裂声

动画:
  - 持续时间: 1s
  - 缓动: ease-in
```

### 6.4 动画时间规范

| 动画类型 | 时长 | 缓动函数 | 说明 |
|----------|------|----------|------|
| 界面切换 | 300ms | ease-out | 快速响应 |
| 按钮悬停 | 150ms | ease | 即时反馈 |
| 地图绘制 | 实时 | linear | 跟随输入 |
| 记忆衰退 | 2000ms | ease-in | 缓慢淡出 |
| 光源闪烁 | 100ms | random | 随机波动 |
| 粒子效果 | 500-2000ms | ease-out | 自然消散 |
| 场景转场 | 500-800ms | ease-in-out | 平滑过渡 |
| 角色移动 | 4帧/步 | linear | 像素动画 |

## 7. 技术规格要求

### 7.1 分辨率与尺寸

#### 游戏画面
| 项目 | 规格 | 说明 |
|------|------|------|
| 基础分辨率 | 1920×1080 | 16:9 标准 |
| 最小分辨率 | 1280×720 | 支持低配置 |
| 最大分辨率 | 3840×2160 | 4K 支持 |
| 像素比例 | 1:1 | 无拉伸 |

#### 精灵尺寸
| 类型 | 尺寸 | 说明 |
|------|------|------|
| 角色 | 32×48 px | 标准比例 |
| 地形单元格 | 32×32 px | 基础单位 |
| 物品图标 | 24×24 px | 背包/工具栏 |
| UI按钮 | 48×48 px | 可点击区域 |
| 地图单元格 | 20×20 px | 地图界面 |

### 7.2 图片格式

| 用途 | 格式 | 压缩 | 说明 |
|------|------|------|------|
| 精灵图 | PNG | 无损 | 保留透明度 |
| 背景图 | JPG/PNG | 质量80% | 大背景图 |
| UI元素 | PNG | 无损 | 清晰边缘 |
| 纹理 | PNG | 无损 | 可平铺 |
| 特效贴图 | PNG | 无损 | 粒子效果 |

### 7.3 动画规格

| 类型 | 帧率 | 格式 | 说明 |
|------|------|------|------|
| 角色动画 | 12 FPS | 精灵表 | 像素风格 |
| UI动画 | 60 FPS | CSS/代码 | 流畅过渡 |
| 特效动画 | 30 FPS | 粒子系统 | 性能平衡 |
| 背景动画 | 15 FPS | 循环 | 节省性能 |

### 7.4 色彩模式

- **主模式**: RGB (屏幕显示)
- **色深**: 8-bit (256色) 或 24-bit (真彩色)
- **色彩空间**: sRGB
- **透明度**: 8-bit Alpha 通道

### 7.5 性能优化

#### 纹理优化
- 使用纹理图集 (Texture Atlas) 减少Draw Call
- 最大纹理尺寸: 2048×2048
- 优先使用 2^n 尺寸 (32, 64, 128, 256...)

#### 动画优化
- 限制同屏粒子数量 (< 100)
- 使用对象池复用粒子
- 远景使用低帧率动画
