# 谜题编辑器使用说明

## 快速开始

### 1. 启动编辑器

```bash
cd tools/level-editor
npm install
npm run dev
```

浏览器会自动打开 `http://localhost:3000`

### 2. 创建第一个关卡

1. 点击顶部"新建"按钮
2. 在属性面板中输入关卡名称
3. 开始编辑

## 功能详解

### 工具栏

位于界面左侧，包含5个主要工具：

| 图标 | 工具 | 快捷键 | 说明 |
|------|------|--------|------|
| ↖ | 选择工具 | 1 | 选择、移动、编辑对象 |
| ▦ | 地形工具 | 2 | 放置地形图块 |
| 🧩 | 谜题工具 | 3 | 放置谜题元素 |
| 🌫 | 迷雾工具 | 4 | 绘制迷雾区域 |
| ⌫ | 橡皮擦 | 5 | 删除对象 |

### 地形编辑

#### 选择图块

1. 点击"地形工具"按钮
2. 在右侧面板选择图块类型：
   - ⬜ 空白 - 删除图块
   - ▦ 地板 - 基础行走区域
   - ▪ 墙壁 - 阻挡移动
   - ≈ 水域 - 不可通行
   - ❀ 草地 - 装饰性地形
   - ◈ 石板 - 硬质地面
   - ▭ 木板 - 木质地板

#### 选择图层

地形分为三个图层：
- **地板层**：最底层，玩家行走的基础
- **墙壁层**：中层，阻挡移动的障碍物
- **装饰层**：最上层，不影响玩法的装饰

#### 放置图块

1. 选择图块类型和图层
2. 在画布上点击放置
3. 按住鼠标拖动可连续放置

### 谜题编辑

#### 谜题类型

| 图标 | 类型 | 说明 |
|------|------|------|
| 🚀 | 起点 | 玩家起始位置（每个关卡只能有一个） |
| 🏁 | 终点 | 关卡目标位置（每个关卡只能有一个） |
| 💎 | 收集物 | 可收集的物品 |
| 🔘 | 开关 | 需要找到正确组合的开关谜题 |
| 🔢 | 序列谜题 | 按正确顺序触发的谜题 |
| ⬛ | 压力板 | 需要按顺序踩下的压力板 |
| 〰 | 路径绘制 | 需要绘制正确路径的谜题 |
| 🔣 | 符号匹配 | 匹配符号的谜题 |
| 💡 | 光线反射 | 调整镜子反射光线的谜题 |
| 🔐 | 组合谜题 | 需要解决多个子谜题的组合 |

#### 放置谜题

1. 点击"谜题工具"按钮
2. 在右侧面板选择谜题类型
3. 在画布上点击放置
4. 选中谜题后可在属性面板编辑详情

#### 编辑谜题属性

选中谜题后，可以编辑：
- **名称**：谜题的显示名称
- **位置**：精确的X/Y坐标
- **提示**：玩家交互时显示的提示文字
- **属性**：根据谜题类型的特殊配置

### 迷雾编辑

#### 笔刷设置

- **笔刷大小**：10-100像素
- **迷雾密度**：0-100%

#### 操作方式

1. 选择"迷雾工具"
2. 调整笔刷大小
3. 在画布上拖动鼠标清除迷雾
4. 使用"填充全部"恢复全迷雾
5. 使用"清除全部"移除所有迷雾

#### 迷雾区域

- 清除的迷雾会显示为透明区域
- 迷雾区域会保存在关卡数据中
- 游戏加载时会自动应用这些区域

### 属性面板

根据当前工具和选中对象显示不同内容：

#### 地形面板
- 图块选择器
- 图层选择

#### 谜题面板
- 谜题类型列表
- 类型说明

#### 迷雾面板
- 笔刷大小滑块
- 迷雾密度滑块
- 快速操作按钮

#### 选中对象面板
- 对象ID
- 名称编辑
- 位置编辑
- 删除按钮

#### 关卡信息面板
- 关卡ID
- 关卡名称
- 描述
- 作者
- 版本

### 顶部操作栏

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| 新建 | 创建空白关卡 | Ctrl+N |
| 打开 | 加载现有关卡 | Ctrl+O |
| 保存 | 保存到本地 | Ctrl+S |
| 导出 | 导出为游戏格式 | Ctrl+E |

### 状态栏

位于界面底部，显示：
- 当前鼠标位置
- 网格尺寸
- 缩放比例
- 操作提示

## 工作流程

### 创建新关卡

1. 点击"新建"按钮
2. 在属性面板设置关卡名称
3. 选择地形工具，绘制基础地形
4. 选择谜题工具，放置起点和终点
5. 添加中间谜题和收集物
6. 选择迷雾工具，设置初始可见区域
7. 点击"导出"生成关卡文件

### 编辑现有关卡

1. 点击"打开"按钮
2. 选择关卡JSON文件
3. 进行需要的修改
4. 点击"保存"或"导出"

### 导出关卡

1. 点击"导出"按钮
2. 选择导出格式：
   - **JSON**：程序化读取的关卡数据
   - **TSCN**：Godot引擎场景文件
3. 选择保存位置
4. 完成导出

## 最佳实践

### 关卡设计

1. **合理布局**：确保起点到终点有明确路径
2. **难度递进**：谜题复杂度逐步增加
3. **迷雾运用**：利用迷雾隐藏部分区域增加探索性
4. **视觉引导**：使用装饰元素引导玩家

### 性能优化

1. **控制图块数量**：避免过多装饰影响性能
2. **合理设置迷雾区域**：不需要精确到像素
3. **及时保存**：防止意外丢失工作

### 团队协作

1. **命名规范**：使用有意义的关卡ID和名称
2. **版本控制**：在描述中记录修改历史
3. **文档注释**：复杂谜题添加详细说明

## 故障排除

### 编辑器无法启动

1. 检查Node.js版本（需要16+）
2. 运行 `npm install` 安装依赖
3. 检查端口3000是否被占用

### 图块无法放置

1. 确认选择了正确的工具
2. 检查是否在画布范围内
3. 查看是否选中了图层

### 导出失败

1. 检查关卡数据是否完整
2. 确认起点和终点已设置
3. 查看浏览器控制台错误信息
