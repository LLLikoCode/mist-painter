# 迷雾绘者 (Mist Painter)

一款冒险解谜游戏，玩家通过绘制迷雾来揭示隐藏的世界。

## 项目信息

- **引擎**: Godot 4.x
- **语言**: GDScript
- **类型**: 冒险解谜
- **版本**: 0.1.0 (开发中)

## 目录结构

```
mist-painter/
├── src/                # 源代码
│   ├── core/          # 核心系统
│   ├── gameplay/      # 游戏逻辑
│   ├── ui/           # 界面系统
│   ├── audio/        # 音频系统
│   ├── save/         # 存档系统
│   └── utils/        # 工具类
├── assets/            # 资源文件
│   ├── sprites/      # 角色精灵
│   ├── tilesets/     # 场景图块
│   ├── ui/          # UI资源
│   ├── audio/       # 音频资源
│   ├── fonts/       # 字体
│   └── effects/     # 特效
├── scenes/           # 场景文件
├── resources/        # 主题、材质等资源
└── tests/           # 测试文件
```

## 核心系统

- **GameStateManager**: 游戏状态管理
- **SceneManager**: 场景切换管理
- **EventBus**: 全局事件系统
- **ConfigManager**: 配置管理
